## v1.5 Requirements

### Bug Fixes

- [x] **BUG-01**: Fix subtraction underflow (3-7 returns 7 instead of wrapping to correct unsigned result) — Fixed in Phase 29
- [x] **BUG-02**: Fix less-or-equal comparison (5<=5 returns 0 instead of 1) — Fixed in Phase 29
- [x] **BUG-03**: Fix multiplication segfaults at certain widths — Fixed in Phase 29
- [x] **BUG-04**: Fix QFT addition bug with both nonzero operands — Fixed in Phase 29

### Verification Framework

- [x] **VFWK-01**: Reusable test framework: build circuit -> export OpenQASM -> Qiskit simulate -> verify outcome — Phase 28
- [x] **VFWK-02**: Parameterized test generation (exhaustive for small widths, representative for larger) — Phase 28
- [x] **VFWK-03**: Clear failure diagnostics (expected vs actual, operation, operand values, bit widths) — Phase 28

### Verification — Initialization

- [x] **VINIT-01**: Verify classical qint initialization produces correct values across all bit widths — Phase 28

### Verification — Arithmetic

- [x] **VARITH-01**: Verify addition (exhaustive 1-4 bits, representative larger) — Phase 30
- [x] **VARITH-02**: Verify subtraction (exhaustive 1-4 bits, representative larger) — Phase 30
- [x] **VARITH-03**: Verify multiplication (exhaustive 1-3 bits, representative larger) — Phase 30
- [x] **VARITH-04**: Verify division and modulo operations — Phase 30
- [x] **VARITH-05**: Verify modular arithmetic (add/sub/mul mod N) — Phase 30

### Verification — Comparison

- [x] **VCMP-01**: Verify all 6 comparison operators (==, !=, <, >, <=, >=) — Phase 31
- [x] **VCMP-02**: Verify qint vs int and qint vs qint comparison variants — Phase 31
- [x] **VCMP-03**: Verify comparison edge cases (equal values, boundaries, zero) — Phase 31

### Verification — Bitwise

- [x] **VBIT-01**: Verify AND, OR, XOR, NOT operations — Phase 32
- [x] **VBIT-02**: Verify bitwise operations with variable-width operands — Phase 32

### Verification — Advanced

- [x] **VADV-01**: Verify automatic uncomputation (ancilla qubits return to |0>) — Phase 33
- [x] **VADV-02**: Verify quantum conditionals (`with` blocks) — Phase 33
- [x] **VADV-03**: Verify ql.array operations (reductions, element-wise) — Phase 33
