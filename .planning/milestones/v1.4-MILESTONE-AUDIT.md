## Tech Debt

No critical blockers. Minor items for tracking:

### Phase 27: Verification Script

1. **3 skipped tests** — Pre-existing C backend bugs prevent testing subtraction underflow, multiplication, and less-or-equal equality. Tests are implemented and documented; they will work when C backend bugs are fixed.
2. **Subprocess overhead** — Each test spawns a fresh Python process (~1.5s each) because `ql.circuit()` doesn't deallocate qubits in-process. Acceptable trade-off for memory safety.
3. **PYTHONPATH requirement** — Development usage outside pytest requires `PYTHONPATH=src`. Standard for in-place development builds.

**Total: 3 items across 1 phase — all non-blocking**
