## Milestone Summary

**Key Decisions:**

- Subprocess isolation for Qiskit verification (C backend doesn't deallocate qubits in-process)
- MSB-first bit extraction for Qiskit result parsing
- Separate result correctness from ancilla cleanup for comparison uncomputation
- Non-strict xfail for discovered bugs (allows xpass without failure)
- Width=3 minimum for meaningful uncomputation tests

**Issues Resolved:**

- BUG-01: Subtraction underflow (3-7 returned 7 instead of wrapping)
- BUG-02: Less-or-equal comparison (5<=5 returned 0)
- BUG-03: Multiplication segfaults at certain widths
- BUG-04: QFT addition with both nonzero operands
- BUG-05: circuit() state reset determinism
- BUG-BIT-01: CQ bitwise bit ordering + mixed-width padding

**Issues Deferred (Tech Debt):**

- BUG-CMP-01: eq/ne return inverted results (488 xfail tests)
- BUG-CMP-02: Ordering comparison errors at MSB boundary
- BUG-CMP-03: Circuit size explosion at widths >= 6 for gt/le
- BUG-DIV-01: Comparison overflow in restoring division
- BUG-DIV-02: MSB comparison leak for large values
- BUG-MOD-REDUCE: _reduce_mod result corruption
- BUG-MOD-SUB: Subtraction extraction position instability
- BUG-COND-MUL-01: Controlled multiplication corruption
- BUG-ARRAY-INIT: Array constructor ignores user values
- CQ mixed-width design limitation (plain int has no width metadata)

**Technical Debt Incurred:**

- 968 xfail tests documenting discovered bugs across 4 phases
- These bugs were *found* through verification — exactly the milestone's purpose
