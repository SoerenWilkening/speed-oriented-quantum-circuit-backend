<task type="auto">
  <name>Task 1: Fix memory leaks in execution.c (stack allocation for gate_t)</name>
  <files>c_backend/src/execution.c</files>
  <action>
Modify `run_instruction()` to use stack-allocated `gate_t` instead of malloc:

**In run_instruction() (around line 26):**
Replace:
```c
gate_t *g = malloc(sizeof(gate_t));
memcpy(g, &res->seq[layer][gate], sizeof(gate_t));
g->Target = qubit_array[g->Target];
```

With:
```c
gate_t g;
memcpy(&g, &res->seq[layer][gate], sizeof(gate_t));
g.Target = qubit_array[g.Target];
```

Update ALL references from `g->` to `g.` in the function body. Change `add_gate(circ, g)` to `add_gate(circ, &g)`.

For n-controlled gates (NumControls > 2), the large_control malloc is still needed because the remapped control array needs to persist until add_gate copies it. But it should be freed after add_gate returns:
```c
if (g.NumControls > 2 && res->seq[layer][gate].large_control != NULL) {
    g.large_control = malloc(g.NumControls * sizeof(qubit_t));
    if (g.large_control != NULL) {
        for (int i = 0; i < (int)g.NumControls; ++i) {
            g.large_control[i] = qubit_array[res->seq[layer][gate].large_control[i]];
        }
        g.Control[0] = g.large_control[0];
        g.Control[1] = g.large_control[1];
    }
} else {
    for (int i = 0; i < (int)g.NumControls && i < MAXCONTROLS; ++i) {
        g.Control[i] = qubit_array[g.Control[i]];
    }
}
```
After `add_gate(circ, &g);`, free the large_control if it was allocated:
```c
add_gate(circ, &g);
if (g.NumControls > 2 && g.large_control != NULL &&
    g.large_control != res->seq[layer][gate].large_control) {
    free(g.large_control);
}
```

Also replace `g->GateValue *= pow(-1, invert);` with `g.GateValue *= (invert ? -1.0 : 1.0);` (micro-optimization: avoid floating-point pow for a trivial computation).

**In reverse_circuit_range() (around line 74):**
Apply the same stack allocation fix:
```c
// Replace:
gate_t *g = malloc(sizeof(gate_t));
if (g == NULL) { return; }
memcpy(g, original_gate, sizeof(gate_t));

// With:
gate_t g;
memcpy(&g, original_gate, sizeof(gate_t));
```

Update all `g->` to `g.` references. Change `add_gate(circ, g)` to `add_gate(circ, &g)`.

Replace `g->GateValue *= pow(-1, 1);` with `g.GateValue = -g.GateValue;` (pow(-1,1) is always -1.0).

For n-controlled gates, same pattern: malloc large_control for remapping, free after add_gate.
After `add_gate(circ, &g);`:
```c
if (g.NumControls > 2 && g.large_control != NULL &&
    g.large_control != original_gate->large_control) {
    free(g.large_control);
}
```

**Important:** The `add_gate()` function already accepts `gate_t*` and copies via `memcpy` in `append_gate()`. Stack-allocated pointers are valid because the data is copied before the pointer goes out of scope.
