<task type="auto">
  <name>Task 1: Create draw.py renderer with layout engine, wires, and gate rendering</name>
  <files>src/quantum_language/draw.py, setup.py</files>
  <action>
Create `src/quantum_language/draw.py` with the following components:

**Constants:**
- `CELL_W = 3`, `CELL_H = 3` (pixels per layer/qubit cell: 2px gate + 1px implicit gap)
- `BG_COLOR = (20, 20, 30)` -- dark background
- `WIRE_COLOR = (60, 60, 80)` -- subtle gray-blue wire
- `CTRL_COLOR = (255, 255, 100)` -- bright yellow for control lines (used in plan 02)
- `GATE_COLORS` dict mapping gate type int to RGB tuple:
  - 0,1,2 (X,Y,Z Pauli): `(180, 130, 200)` muted purple
  - 3,5,6,7 (R,Rx,Ry,Rz rotation): `(130, 180, 200)` muted cyan
  - 4 (H Hadamard): `(200, 180, 100)` warm yellow -- distinct
  - 8 (P Phase): `(100, 200, 150)` muted green
  - 9 (M Measurement): `(200, 100, 100)` muted red

**`render(draw_data)` function:**
1. Extract `num_layers` and `num_qubits` from draw_data
2. Handle empty circuit: if either is 0, return `Image.fromarray(np.full((1, 1, 3), BG_COLOR, dtype=np.uint8), 'RGB')`
3. Compute `width = num_layers * CELL_W`, `height = num_qubits * CELL_H`
4. Allocate canvas: `np.full((height, width, 3), BG_COLOR, dtype=np.uint8)`
5. Draw horizontal wires: for each qubit q, `wire_y = q * CELL_H + CELL_H // 2`, set `canvas[wire_y, :] = WIRE_COLOR`
6. (Control lines placeholder -- implemented in plan 02)
7. Draw gate blocks: for each gate in `draw_data['gates']`:
   - `gx = gate['layer'] * CELL_W`, `gy = gate['target'] * CELL_H`
   - If gate type == 9 (M): render checkerboard -- `canvas[gy, gx] = color; canvas[gy+1, gx+1] = color` (leave other 2 pixels as BG)
   - Else: `canvas[gy:gy+2, gx:gx+2] = GATE_COLORS.get(gate['type'], (150, 150, 150))`
8. (Control dots placeholder -- implemented in plan 02)
9. Return `Image.fromarray(canvas, 'RGB')`

**IMPORTANT implementation notes:**
- Use `import numpy as np` and `from PIL import Image` at top
- Array indexing is `canvas[y, x]` (row, col) -- NOT (x, y)
- Use only NumPy slice assignment for bulk rendering (satisfies REND-05)
- No ImageDraw calls anywhere
- The function should accept an optional `cell_size` parameter (default 3) to allow flexibility

**setup.py change:**
Add `install_requires=["Pillow>=9.0"]` to the `setup()` call. Place it after `package_data` and before `extras_require`.
