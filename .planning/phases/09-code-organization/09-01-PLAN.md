<objective>
Create a dedicated arithmetic operations header that consolidates addition and multiplication function declarations.

Purpose: Establish a clear module for arithmetic operations (addition, multiplication) as the first step toward CODE-04 compliance.

Note on subtraction: The C layer has NO dedicated subtraction functions (QQ_sub, CQ_sub do not exist). Subtraction is implemented at the Python level using addition with two's complement negation - i.e., `a - b` uses `QQ_add(bits)` with the second operand negated via `two_complement()`. This is standard quantum arithmetic practice. The arithmetic_ops.h header therefore covers addition and multiplication only; subtraction capability comes from the addition functions.

Output: arithmetic_ops.h header file with all arithmetic function declarations, Integer.h updated to include it.
</objective>
