# Quantum Assembly

Write quantum algorithms in natural programming style that compiles to efficient, memory-optimized quantum circuits.

## Quick Start

```python
from quantum_language import qint, qbool, circuit

# Create circuit and quantum integers
c = circuit()
a = qint(5, width=8)   # 8-bit quantum integer, value 5

# Perform arithmetic (generates quantum circuit)
result = a + b

# View circuit statistics
print(f"Gates: {c.gate_count}, Depth: {c.depth}")

# Export to OpenQASM
c.visualize()
```

## API Reference

### qint - Quantum Integer

```python
qint(value=0, width=8)
```

Create a quantum integer.

**Parameters:**
- `value` (int): Initial classical value (default 0)
- `width` (int): Bit width, 1-64 (default 8)
- `bits` (int): Alias for width (backward compatibility)

**Properties:**
- `width`: Bit width (read-only)

**Arithmetic Operations:**
| Operation | Example | Returns |
|-----------|---------|---------|
| Addition | `a + b`, `a + 5` | qint |
| Subtraction | `a - b`, `a - 5` | qint |
| Multiplication | `a * b`, `a * 5` | qint |
| Floor Division | `a // b`, `a // 5` | qint |
| Modulo | `a % b`, `a % 5` | qint |
| Divmod | `divmod(a, 5)` | (qint, qint) |

**Bitwise Operations:**
| Operation | Example | Returns |
|-----------|---------|---------|
| AND | `a & b`, `a & 5` | qint |
| OR | `a \| b`, `a \| 5` | qint |
| XOR | `a ^ b`, `a ^ 5` | qint |
| NOT | `~a` | qint |

**Comparison Operations:**
| Operation | Example | Returns |
|-----------|---------|---------|
| Equal | `a == b` | qbool |
| Not Equal | `a != b` | qbool |
| Less Than | `a < b` | qbool |
| Greater Than | `a > b` | qbool |
| Less/Equal | `a <= b` | qbool |
| Greater/Equal | `a >= b` | qbool |

**Quantum Conditionals:**
```python
condition = a < 10
with condition:
    a += 5  # Only executes if condition is True
```

### qbool - Quantum Boolean

```python
qbool(value=False)
```

1-bit quantum integer (inherits from qint).

**Parameters:**
- `value` (bool): Initial value (default False)

### qint_mod - Modular Arithmetic

```python
qint_mod(value, N)
```

Quantum integer with automatic modular reduction.

**Parameters:**
- `value` (int): Initial value (reduced mod N)
- `N` (int): Modulus (required, positive integer)

**Properties:**
- `modulus`: The modulus N (read-only)

**Operations:** Same as qint, all results reduced mod N.

### circuit - Circuit Manager

```python
circuit()
```

Singleton circuit manager (created automatically).

**Properties:**
- `gate_count`: Total gates in circuit
- `depth`: Circuit depth (parallel layers)
- `qubit_count`: Qubits used
- `gate_counts`: Dict with gate type breakdown
- `available_passes`: List of optimization pass names

**Methods:**
- `visualize()`: Print ASCII circuit visualization
- `optimize(passes=None)`: Run optimization, return stats
- `can_optimize()`: Check if optimization would help

## Examples

### Quantum Arithmetic

```python
# 16-bit addition
a = qint(100, width=16)
b = qint(200, width=16)
c = a + b

# Mixed-width (result uses max width)
small = qint(5, width=8)
large = qint(1000, width=16)
result = small + large  # result.width == 16
```

### Quantum Comparisons

```python
a = qint(42, width=8)

# Comparison returns qbool
is_big = a > 100
is_equal = a == 42

# Use in conditional
with is_big:
    a -= 50  # Controlled subtraction
```

### Modular Arithmetic (for Cryptography)

```python
# RSA-style modular operations
N = 17
x = qint_mod(5, N=N)
y = qint_mod(12, N=N)

# Automatic reduction mod N
z = x * y  # (5 * 12) mod 17 = 9
```

### Circuit Optimization

```python
c = circuit()
a = qint(5, width=8)
b = qint(3, width=8)

# Build circuit
for _ in range(10):
    a += b
    a -= b  # Cancels with previous add

# Optimize
stats = c.optimize()
print(f"Before: {stats['before']['gate_count']}")
print(f"After: {stats['after']['gate_count']}")  # Much smaller
```

### Array Operations

```python
# Create array of quantum integers
qubits = array(10)  # 10 qints with default width

# Initialize from values
values = array([1, 2, 3, 4, 5])

# 2D array
matrix = array((3, 3))  # 3x3 array of qints
```
