<objective>
Optimize qarray element-wise operations with classical (int/list) operands to eliminate temporary qint allocations per element by passing int values directly to qint's in-place operators, which already handle int natively via CQ_* C backend calls.

Purpose: Reduce circuit-generation overhead for array operations with classical operands. For the scalar int case, the current code creates ONE temporary qint (line 868) and reuses it for all elements (line 870) -- a minor allocation. For the list-of-int case, the current code creates a temporary qint PER element inside the loop (line 885) -- the real per-element allocation overhead. Since qint's in-place operators (e.g. `__iadd__`, `__imul__`) already check for int operands and call CQ_* directly, both temporary qint allocations are unnecessary.

Output: Modified `qarray.pyx` with `_inplace_binary_op` that passes int operands directly to qint operators without wrapping them in temporary qint objects.
</objective>
