```python
@ql.compile(inverse=True)
def attack():
    arr = ql.qarray([[0] * 8] * 8, width=3)
    for i, j in itertools.product(...):
        with knights[ord(i) - ord("a"), j]:
            arr += knight_moves((i, j))
    attacked = arr != 0
    king = ql.qarray([[0] * 8] * 8, dtype=ql.qbool)
    in_check = (king & attacked).any()
    return in_check
```

IMPORTANT framework constraints:
- `with qbool:` blocks CANNOT nest (known limitation -- use precomputed combined conditions)
- @ql.compile(inverse=True) requires all operations inside to be reversible
- Classical precomputation happens OUTSIDE the @ql.compile function body
- Qiskit spot-checks must stay within 17-qubit budget (max_parallel_threads=4)
