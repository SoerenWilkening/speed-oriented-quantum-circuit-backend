```python
def counting_diffusion_core(
    validity, d_max, branch_reg, h_qubit_idx, h_child_idx,
    variable_angles, variable_root_angles, is_root
):
    """Core counting-based diffusion -- shared by QWalkTree and chess_walk.

    Parameters
    ----------
    validity : list[qbool]
        Validity ancillae, one per potential child.
    d_max : int
        Maximum branching factor.
    branch_reg : qint
        Branch register for this level.
    h_qubit_idx : int
        Physical qubit index for h[depth].
    h_child_idx : int
        Physical qubit index for h[depth-1].
    variable_angles : dict
        Pre-computed angles from _plan_cascade_ops / precompute_diffusion_angles.
    variable_root_angles : dict
        Root-level angle overrides (empty dict if not root).
    is_root : bool
        Whether this is the root depth.
    """
```

This function contains the counting logic (sum, dispatch loop, S_0, forward, uncompute) currently in `_counting_diffusion`. Make `QWalkTree._counting_diffusion` delegate to this standalone function.
