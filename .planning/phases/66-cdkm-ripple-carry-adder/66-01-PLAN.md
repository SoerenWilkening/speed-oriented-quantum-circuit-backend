**2. Modify hot_path_add_qq():**

After the existing `int result_bits` computation and qa[] population of self and other qubits, add a Toffoli dispatch branch:

```c
if (circ->arithmetic_mode == ARITH_TOFFOLI && !controlled) {
    // Toffoli CDKM path
    sequence_t *seq;
    if (result_bits == 1) {
        // 1-bit: no ancilla needed
        seq = toffoli_QQ_add(result_bits);
        if (seq == NULL) return;
        run_instruction(seq, qa, invert, circ);
    } else {
        // n >= 2: allocate 1 ancilla qubit
        qubit_t ancilla_qubit = allocator_alloc(circ->allocator, 1, /*is_ancilla=*/1);
        if (ancilla_qubit == (qubit_t)-1) return;  // allocation failed

        // Ancilla goes at virtual index 2*result_bits (after both registers)
        qa[2 * result_bits] = ancilla_qubit;

        seq = toffoli_QQ_add(result_bits);
        if (seq == NULL) {
            allocator_free(circ->allocator, ancilla_qubit, 1);
            return;
        }
        run_instruction(seq, qa, invert, circ);

        // Free ancilla (CDKM guarantees return to |0>)
        allocator_free(circ->allocator, ancilla_qubit, 1);
    }
    return;
}

// Toffoli mode with controlled: fall back to QFT
// (controlled Toffoli adder is Phase 67)
```

Place this block BEFORE the existing controlled/uncontrolled QFT dispatch. If arithmetic_mode is ARITH_TOFFOLI and controlled, fall through to the existing QFT path.

**3. Modify hot_path_add_cq():**

if (circ->arithmetic_mode == ARITH_TOFFOLI && !controlled) {

    if (self_bits == 1) {

        seq = toffoli_CQ_add(self_bits, classical_value);
        if (seq == NULL) {

**CQ qubit layout contract for Toffoli:** The CQ sequence expects `[0..bits-1]` = target, `[bits]` = ancilla. The hot path maps `qa[0..self_bits-1]` = self_qubits, `qa[self_bits]` = ancilla_qubit. This matches the CQ sequence layout.

**QQ qubit layout contract for Toffoli:** The QQ sequence expects `[0..bits-1]` = register a, `[bits..2*bits-1]` = register b, `[2*bits]` = ancilla. The hot path already populates qa[0..self_bits-1] = self qubits and qa[self_bits..self_bits+other_bits-1] = other qubits. For mixed-width, result_bits = max(self_bits, other_bits), and the sequence operates on result_bits. The qa[] mapping MUST have `result_bits` entries for each register. The existing QFT path works because the Cython layer ensures both registers have been extended to result_bits before calling the hot path. The Toffoli path relies on the same contract.
