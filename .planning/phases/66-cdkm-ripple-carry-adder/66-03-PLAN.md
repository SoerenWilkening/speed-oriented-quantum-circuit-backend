Fix CQ Toffoli addition by replacing the buggy 2-qubit MAJ/UMA CQ simplification with a correct temp-register approach that reuses the proven QQ CDKM adder.

Purpose: Close the gap in Success Criterion 2 (CQ addition widths 2+).   **Step 2: Rewrite the general case (bits >= 2) of toffoli_CQ_add.**

  New qubit layout (documented in function header):
  ```
  [0..bits-1]       = temp register (a-register in CDKM, initialized to classical value)
  [bits..2*bits-1]  = self register (b-register in CDKM, receives sum)
  [2*bits]          = carry ancilla
  ```

  New implementation for the general case:
  1. Call two_complement(value, bits) to get binary representation (existing code, keep as-is).
  2. Count x_count = popcount of the classical value (number of 1-bits in bin[]).
  3. Compute num_layers = 2 * x_count + 6 * bits (x_count X-gates for init, 6*bits for QQ CDKM, x_count X-gates for cleanup).
  4. Allocate sequence with alloc_sequence(num_layers).
  5. Set carry = 2 * bits.
  6. **Phase 1 - X-init temp register:** For each bit i from 0 to bits-1, if bin[bits-1-i] == 1 (i.e., LSB-first classical bit is 1), emit X gate on qubit i. Each X gate gets its own layer.
  7. **Phase 2 - QQ CDKM adder:** Reuse the exact same MAJ/UMA chain as toffoli_QQ_add:
     - emit_MAJ(seq, &layer, carry, bits + 0, 0)
     - for i=1..bits-1: emit_MAJ(seq, &layer, i-1, bits + i, i)
     - for i=bits-1..1: emit_UMA(seq, &layer, i-1, bits + i, i)
     - emit_UMA(seq, &layer, carry, bits + 0, 0)
  8. **Phase 3 - X-cleanup temp register:** Same X gates as Phase 1 (CDKM preserves a-register, so temp is still the classical value; X is self-inverse, so applying the same X gates resets temp to |0>).
  9. Set seq->used_layer = layer.
  10. Free bin, return seq.

  **Keep the 1-bit special case (lines 300-327) completely unchanged** -- it already works correctly.

  **Keep the bounds check (bits < 1 || bits > 64) unchanged.**

  **Update the function header comment** to document the new qubit layout:
  ```
  // Qubit layout for toffoli_CQ_add(bits, value):
  //   [0..bits-1]       = temp register (initialized to classical value, cleaned to |0>)
  //   [bits..2*bits-1]  = self register (target, modified: self += value)
  //   [2*bits]          = carry ancilla (bits >= 2 only)
  ```

  **Why this works:**
  - The CDKM adder computes b = a + b and preserves a. Since temp is initialized to the classical value and the adder preserves it, temp still holds the classical value after CDKM. The same X gates that initialized it also undo it.
  - Inversion (subtraction) works because: X is self-inverse (X-init and X-cleanup commute with inversion), and running the CDKM sequence in reverse computes subtraction.
  - The existing emit_MAJ and emit_UMA functions are proven correct by exhaustive QQ tests.

  **Do NOT modify:** emit_MAJ, emit_UMA, alloc_sequence, toffoli_QQ_add, toffoli_sequence_free. These are all correct and well-tested.
