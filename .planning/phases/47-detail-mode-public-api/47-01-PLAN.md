Add a `render_detail()` function to `draw.py` that renders circuits at detail zoom level. Specifics:

**Constants and setup:**
- `DETAIL_CELL = 12` (12px per cell — accommodates "Rx" at font size 8 which is 11px wide)
- `LABEL_MARGIN = 40` (left margin for qubit labels like "q0", "q1", etc.)
- Gate type integer-to-label mapping: `GATE_LABELS = {0: "X", 1: "Y", 2: "Z", 3: "R", 4: "H", 5: "Rx", 6: "Ry", 7: "Rz", 8: "P"}` (no entry for 9/measurement — it uses checkerboard icon)
- `BORDER_COLOR = (80, 80, 100)` for 1px gate box borders
- Use `from PIL import ImageDraw, ImageFont` (already acceptable since detail mode targets small circuits)

**Function signature:** `def render_detail(draw_data):`

**Canvas:** Width = `LABEL_MARGIN + num_layers * DETAIL_CELL`, Height = `num_qubits * DETAIL_CELL`. Fill with BG_COLOR. Create ImageDraw.Draw on the PIL Image.

**Pre-compute measurement positions:** Build a set of `(layer, target)` tuples for all measurement gates (type==9). For each qubit, find the earliest measurement layer (if any) to know where to stop wires.

**Step 1 — Qubit labels:** For each qubit q, draw text `f"q{q}"` at x=2, y = `q * DETAIL_CELL + (DETAIL_CELL - text_height) // 2`, using white text `(200, 200, 200)`.

**Step 2 — Horizontal wires:** For each qubit q, draw a horizontal line at y = `q * DETAIL_CELL + DETAIL_CELL // 2` from x = `LABEL_MARGIN` to either the end of the canvas or just before the earliest measurement gate on that qubit (x = `LABEL_MARGIN + earliest_meas_layer * DETAIL_CELL + DETAIL_CELL - 1`). Use WIRE_COLOR. Draw using `draw.line()`.

**Step 3 — Control lines:** For each gate with controls, compute x_center = `LABEL_MARGIN + gate_layer * DETAIL_CELL + DETAIL_CELL // 2`. Compute y_min and y_max from all involved qubits (target + controls) at their wire centers. Draw vertical line from y_min to y_max using CTRL_COLOR with `draw.line()`.

**Step 4 — Gate blocks:** For each gate:
- Compute `gx = LABEL_MARGIN + gate_layer * DETAIL_CELL`
- Compute `gy = gate_target * DETAIL_CELL`
- If type == 9 (measurement): draw scaled-up checkerboard pattern. Fill the cell with alternating 2x2 pixel blocks of GATE_COLORS[9] and BG_COLOR in a checkerboard pattern. Do NOT draw "M" text.
- Else: draw filled rectangle `[gx+1, gy+1, gx+DETAIL_CELL-2, gy+DETAIL_CELL-2]` with gate color fill and BORDER_COLOR outline. Then center the gate label text (from GATE_LABELS) inside the box. Use `font = ImageFont.load_default(size=8)`. Compute text bounding box with `font.getbbox(label)`, center it in the cell, draw with white text `(255, 255, 255)`.

**Step 5 — Control dots:** For each gate with controls, draw filled circles (using `draw.ellipse()`) at each control qubit's wire center position. Circle radius = 3px. Fill with CTRL_COLOR.

**Return:** The PIL Image.

**Font:** Use `ImageFont.load_default(size=8)`. This is available in Pillow >= 10.0.

**Wire thickness:** Use width=1 for wires at detail scale (default for draw.line).
