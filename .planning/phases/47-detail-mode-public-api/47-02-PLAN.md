```python
def draw_circuit(circuit, *, mode=None, save=None):
```

**Function logic:**
1. Call `draw_data = circuit.draw_data()` to extract circuit data.
2. Read `num_qubits` and `num_layers` from draw_data.
3. **Auto-zoom logic** (when `mode is None`):
   - If `num_qubits <= AUTO_DETAIL_MAX_QUBITS` AND `num_layers <= AUTO_DETAIL_MAX_LAYERS`: select detail mode. Print: `f"Auto-selected detail mode ({num_qubits} qubits, {num_layers} layers)"`
   - Otherwise: select overview mode. Print: `f"Auto-selected overview mode ({num_qubits} qubits, {num_layers} layers)"`
4. **Mode override** (when `mode` is specified):
   - Validate: must be `"overview"` or `"detail"`. Raise `ValueError(f"mode must be 'overview' or 'detail', got '{mode}'")` otherwise.
   - If `mode == "detail"` and (`num_qubits > AUTO_DETAIL_MAX_QUBITS` or `num_layers > AUTO_DETAIL_MAX_LAYERS`): print warning `f"Warning: detail mode requested for large circuit ({num_qubits} qubits, {num_layers} layers). Rendering may be slow."`, then use detail mode.
   - Otherwise use the requested mode.
5. **Render:** Call `render(draw_data)` for overview or `render_detail(draw_data)` for detail.
6. **Save:** If `save` is not None, call `img.save(save)`.
7. **Return:** The PIL Image.

**In __init__.py — lazy import of draw_circuit:**

Do NOT add `from .draw import draw_circuit` at the top level (this would fail if Pillow is missing since draw.py imports PIL at top level).

Instead, add a wrapper function:

```python
def draw_circuit(circuit, *, mode=None, save=None):
    """Visualize a quantum circuit as a pixel-art image.

    Parameters
    ----------
    circuit : circuit
        A built quantum circuit object.
    mode : str, optional
        Zoom mode: "overview" (compact), "detail" (readable labels),
        or None for auto-selection based on circuit size.
    save : str, optional
        If provided, save the image to this file path.

    Returns
    -------
    PIL.Image.Image
        Rendered circuit image.

    Raises
    ------
    ImportError
        If Pillow is not installed.
    """
    try:
        from .draw import draw_circuit as _draw_circuit
    except ImportError:
        raise ImportError(
            "Pillow is required for circuit visualization. "
            "Install with: pip install Pillow"
        )
    return _draw_circuit(circuit, mode=mode, save=save)
```

Add `"draw_circuit"` to the `__all__` list (in the "Utilities" section).

**Important:** The existing `from PIL import Image` at the top of `draw.py` stays as-is. The lazy import is only in `__init__.py` — `draw.py` itself is only imported when `draw_circuit()` is called.
