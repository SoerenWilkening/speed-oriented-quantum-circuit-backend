<objective>
Remove purely classical functions that only write to QPU_state registers without generating quantum gates.

Purpose: These functions (CC_add, CC_mul, CC_equal, not_seq, and_seq, xor_seq, or_seq) perform classical computations by writing to QPU_state->R0/R1/R2 but don't generate any quantum gates. They are legacy code that was replaced by width-parameterized functions with explicit parameters. Removing them eliminates dead code and removes dependencies on QPU_state.

Output: C files with classical-only functions removed, compilation verified.
</objective>
