Remove the QPU_state global variable, instruction_list array, instruction_t type, and all infrastructure that depended on them.

Purpose: With all functions now using explicit parameters instead of global state, the QPU_state infrastructure is no longer needed. Removing it eliminates the last global state from the C backend and simplifies the architecture.

Output: Clean C backend with no global state. QPU.h becomes a thin backward-compatibility wrapper around circuit.h.
</objective>

  <name>Task 4: Refactor main.c to not depend on QPU_state</name>

  <action>
Refactor main.c to work without QPU_state global state.

**Current main.c (lines 27-38) uses:**

- `QPU_state->R0 = malloc(sizeof(int));` (line 28)
- `*QPU_state->R0 = 3;` (line 29)
- `qubit_mapping(qubit_array, circ);` (line 37) - this is being removed from execution.c

**Refactor to:**
1. Remove all QPU_state references
2. Remove instruction_counter initialization (line 10)
3. Remove the call to qubit_mapping() - initialize qubit_array directly
4. Keep using qq_or_seq() which doesn't need QPU_state (it doesn't read global state)

**New main.c structure:**
```c
#include "circuit.h"
#include "LogicOperations.h"
#include "execution.h"
#include <stdio.h>
#include <time.h>

int main(int argc, char *argv[]) {
    int num_qubits = 64;
    int run = 1;

    if (argc > 2) {
        num_qubits = (int)strtol(argv[1], NULL, 10);
        run = (int)strtol(argv[2], NULL, 10);
    }

    clock_t t1 = clock();
    sequence_t *seq;
    clock_t t2 = clock();

    if (run) {
        // Generate sequence (qq_or_seq doesn't use QPU_state)
        seq = qq_or_seq();

        // Initialize circuit and qubit array directly (no QPU_state)
        circuit_t *circ = init_circuit();
        qubit_t qubit_array[6 * INTEGERSIZE];
        for (int i = 0; i < 6 * INTEGERSIZE; i++) {
            qubit_array[i] = i;
        }

        run_instruction(seq, qubit_array, true, circ);
        print_circuit(circ);

        printf("%f\n", (double)(clock() - t1) / CLOCKS_PER_SEC);
        free_circuit(circ);
    } else {
        printf("%f\n", (double)(t2 - t1) / CLOCKS_PER_SEC);
    }

    free(seq);
    return 0;
}
```

Remove includes for AssemblyOperations.h and AssemblyReader.h if no longer needed.
