   a. Add `_compute_attack_table(moving_piece_type, dr, df, board_rows, board_cols, enemy_attacks)` helper:
      - For each possible king position (kr, kf) on board:
        - If moving_piece_type == "king": check_square = (kr+dr, kf+df). Skip if off-board.
        - Else: check_square = (kr, kf).
        - For each (enemy_idx, (offsets, label)) in enemy_attacks: for each (adr, adf) in offsets, compute attacker square (check_r+adr, check_f+adf). If in bounds, add (ar, af, enemy_idx) to attackers list.

      - Returns the table.

   b. Add `make_check_detection_predicate(moving_piece_type, dr, df, board_rows, board_cols, enemy_attacks)`:
      - Call `_compute_attack_table` for classical precomputation.
      - Define inner `@ql.compile(inverse=True) def check_safe(king_qarray, *args)`:
        - Parse args: `enemy_qarrays = args[:-1]`, `result = args[-1]`.
        - For each (kr, kf, attackers) in attack_table:
          - `with king_qarray[kr, kf]: ~result` (optimistic flip: assume safe).
          - For each (ar, af, enemy_idx) in attackers:
            - Allocate `enemy_flag = ql.qbool()`.
            - `with enemy_qarrays[enemy_idx][ar, af]: ~enemy_flag` (detect attacker).
            - `cond = king_qarray[kr, kf] & enemy_flag` (Toffoli AND).
            - `with cond: ~result` (un-flip: king in check).
            - `with enemy_qarrays[enemy_idx][ar, af]: ~enemy_flag` (uncompute enemy_flag).
      - Return check_safe.
