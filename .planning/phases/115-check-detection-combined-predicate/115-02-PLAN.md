      - result (1 qubit)
      - Total data: 13 qubits + ancillas

   a. Add `make_combined_predicate(piece_type, dr, df, board_rows, board_cols, enemy_attacks, n_friendly, n_enemy)`:
      - `n_friendly`: number of friendly board qarrays (known at construction time)
      - `n_enemy`: number of enemy board qarrays (known at construction time)
      - At construction time, build all three sub-predicates:
        ```python
        pe_pred = make_piece_exists_predicate(piece_type, dr, df, board_rows, board_cols)
        nfc_pred = make_no_friendly_capture_predicate(dr, df, board_rows, board_cols)
        cd_pred = make_check_detection_predicate(piece_type, dr, df, board_rows, board_cols, enemy_attacks)
        ```

      - Define inner `@ql.compile(inverse=True) def is_legal(piece_qarray, *args)`:
        - Parse args using captured n_friendly, n_enemy counts:
          ```python
          friendly_qarrays = args[:n_friendly]
          king_qarray = args[n_friendly]
          enemy_qarrays = args[n_friendly + 1 : n_friendly + 1 + n_enemy]
          result = args[-1]
          ```
        - Allocate 3 intermediate qbools: `pe_result`, `nfc_result`, `cd_result`.
        - Call sub-predicates:
          ```python
          pe_pred(piece_qarray, pe_result)
          nfc_pred(piece_qarray, *friendly_qarrays, nfc_result)
          cd_pred(king_qarray, *enemy_qarrays, cd_result)
          ```
        - Three-way AND:
          ```python
          ab = pe_result & nfc_result
          abc = ab & cd_result
          with abc:
              ~result
          ```
        - Uncompute sub-predicate results (reverse order):
          ```python
          cd_pred.adjoint(king_qarray, *enemy_qarrays, cd_result)
          nfc_pred.adjoint(piece_qarray, *friendly_qarrays, nfc_result)
          pe_pred.adjoint(piece_qarray, pe_result)
          ```
      - Return is_legal.

      IMPORTANT OPEN QUESTION: Nested compiled function calls (calling @ql.compile functions inside another @ql.compile body) may or may not work. Test this EARLY in the implementation:
      1. First attempt: Call sub-predicates directly as shown above.
      2. If framework raises error or produces incorrect gates: INLINE the sub-predicate logic (copy the loop structures from make_piece_exists_predicate, make_no_friendly_capture_predicate, and make_check_detection_predicate directly into the is_legal body). This is more verbose but guaranteed to work since it avoids nested compilation.
      3. Document which approach was used in the SUMMARY.
