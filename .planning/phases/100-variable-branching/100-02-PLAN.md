**IMPORTANT:** The exact predicate implementation depends on the framework's state when called. The predicate sees the full tree state via `node.branch_values` and `node.depth`. A simple test predicate should:
1. Look at a specific branch register value to decide rejection
2. Use CNOT from the branch qubit to the is_reject qbool

class TestVariableBranchingAmplitudes:
    """DIFF-04: Amplitude verification for variable branching."""

    def test_amplitude_d1_from_pruning(self):
        """Node with d(x)=1 (one child rejected) uses phi = 2*arctan(1) = pi/2.

        When only one child is valid, the parent-child split uses phi=pi/2
        giving amplitudes cos(pi/4) = sin(pi/4) = 1/sqrt(2) for parent
        and the single valid child.

        Tree: max_depth=2, branching=2 with predicate rejecting child 1.
        Total qubits <= 17."""

    def test_amplitude_d2_all_valid(self):
        """Node with d(x)=2 (all valid) matches uniform phi = 2*arctan(sqrt(2)).

        Tree: max_depth=2, branching=2 with always-accept predicate.
        Total qubits <= 17."""

    def test_differential_branching_different_amplitudes(self):
        """Tree with d(x)=1 at one node and d(x)=2 at another has different amplitudes.

        This is THE key success criterion: different nodes get different

        Tree: max_depth=2, branching=2, predicate rejects one child at depth=1
        but accepts all at depth=2 (root). Total qubits <= 17.

**Key test: test_differential_branching_different_amplitudes:**

This test is the core success criterion from the roadmap. It must demonstrate that:
1. A tree with a pruning predicate produces DIFFERENT amplitudes at a pruned node vs an unpruned node
2. The amplitudes match the theoretical values for their respective d(x) counts

The test creates two circuits:
- Circuit A: Tree with predicate that rejects child 1 at depth 1 -> d(x)=1 at depth 1
- Circuit B: Tree with no predicate (or always-accept) -> d(x)=2 at depth 1

Compare the statevectors: they should differ at the amplitudes corresponding to child states, matching their respective phi values.
