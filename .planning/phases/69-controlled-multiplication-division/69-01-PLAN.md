Add two new functions to ToffoliMultiplication.c:

**1. `toffoli_cmul_qq`** -- Controlled QQ multiplication using AND-ancilla pattern:

Signature: `void toffoli_cmul_qq(circuit_t *circ, const unsigned int *ret_qubits, int ret_bits, const unsigned int *self_qubits, int self_bits, const unsigned int *other_qubits, int other_bits, unsigned int ext_ctrl)`

Algorithm (n = ret_bits, loop j = 0..n-1, width = n - j):
- Before the loop: allocate 1 carry ancilla and 1 AND ancilla via `allocator_alloc(circ->allocator, 1, true)`. Keep both allocated for the entire loop (reused each iteration since CDKM guarantees return to |0>).
- For width == 1 (j == n-1): Emit a single MCX with 3 controls: `mcx(g, ret_qubits[n-1], controls=[self_qubits[0], other_qubits[j], ext_ctrl], 3)`. Use `add_gate(circ, &g)` to emit directly. The MCX needs a `large_control` array allocated via malloc (3 * sizeof(qubit_t)) since mcx() stores the pointer. The circuit takes ownership of large_control (do NOT free it -- same pattern as run_instruction from Phase 67-02 MCX fix).
- For width >= 2:
  a. Compute AND: Build a gate_t for CCX, call `ccx(&g, and_ancilla, other_qubits[j], ext_ctrl)`, then `add_gate(circ, &g)`.
  b. Build tqa array:
     - tqa[0..width-1] = self_qubits[0..width-1] (a-register, source, preserved)
     - tqa[width..2*width-1] = ret_qubits[j..j+width-1] (b-register, accumulator)
     - tqa[2*width] = carry_ancilla
     - tqa[2*width+1] = and_ancilla (acts as "control" for cQQ_add)
  c. Call `toffoli_cQQ_add(width)` to get the sequence, then `run_instruction(toff_seq, tqa, 0, circ)`.
  d. Uncompute AND: Same CCX as step (a): `ccx(&g, and_ancilla, other_qubits[j], ext_ctrl)`, then `add_gate(circ, &g)`.
- After the loop: free both ancilla via `allocator_free(circ->allocator, carry, 1)` and `allocator_free(circ->allocator, and_anc, 1)`.

IMPORTANT: For the `gate_t` struct, zero-initialize with `memset(&g, 0, sizeof(gate_t))` before each ccx()/mcx() call. The ccx() function from gate.h sets GateType, target, control, and control2. The mcx() function from gate.h sets GateType, target, large_control, and num_controls. For MCX with 3 controls, allocate `large_control = malloc(3 * sizeof(qubit_t))` and pass it to mcx(). Do NOT free this pointer -- the circuit takes ownership via add_gate (same as the MCX ownership fix in Phase 67-02).

**2. `toffoli_cmul_cq`** -- Controlled CQ multiplication:

Signature: `void toffoli_cmul_cq(circuit_t *circ, const unsigned int *ret_qubits, int ret_bits, const unsigned int *self_qubits, int self_bits, int64_t classical_value, unsigned int ext_ctrl)`

Algorithm (n = ret_bits):
- Convert classical_value to binary: `int *bin = two_complement(classical_value, n)`.
- Allocate 1 carry ancilla before the loop.
- For each j (0..n-1) where bin[n-1-j] == 1 (set bit at weight 2^j):
  - width = n - j.
  - For width == 1: Use toffoli_cQQ_add(1) with tqa = [ret_qubits[n-1], self_qubits[0], ext_ctrl]. This emits a CCX (controlled add of 1 bit conditioned on ext_ctrl).
  - For width >= 2: Build tqa as:
    - tqa[0..width-1] = self_qubits[0..width-1]
    - tqa[width..2*width-1] = ret_qubits[j..j+width-1]
    - tqa[2*width] = carry
    - tqa[2*width+1] = ext_ctrl
  - Call `toffoli_cQQ_add(width)`, then `run_instruction(toff_seq, tqa, 0, circ)`.
- Free carry ancilla. Free bin.

Key difference from uncontrolled CQ: uncontrolled uses `toffoli_QQ_add` (no control). Controlled uses `toffoli_cQQ_add` with ext_ctrl mapped to the control slot.
