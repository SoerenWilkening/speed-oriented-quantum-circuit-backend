Replace the temp-register CQ/cCQ CDKM and BK CLA adder implementations in ToffoliAddition.c with inline generators that exploit known classical bit values to simplify gates at generation time. Write exhaustive correctness and gate reduction tests.

Purpose: Reduce T-gate count in CQ and cCQ Toffoli arithmetic by eliminating gates controlled on known-|0> qubits and simplifying gates controlled on known-|1> qubits. The cCQ variant benefits most: 14T saved per zero-bit position in the classical value.

Output: Modified ToffoliAddition.c with 4 inline generators (CDKM CQ, CDKM cCQ, BK CQ, BK cCQ) and a comprehensive test file.
</objective>

  <action>
Replace the bodies of 4 functions in ToffoliAddition.c with inline generators that apply classical-bit gate simplification:

**1. `toffoli_CQ_add(bits, value)` — Inline CDKM CQ generator:**

Add two static helper functions:
- `emit_CQ_MAJ(seq, layer, a, b, c, classical_bit_c, a_known_zero)` — emits simplified MAJ gates:
  - If classical_bit_c == 0: skip 2 CX (control is |0>), emit CCX unless a_known_zero (then entire MAJ is NOP since CCX has |0> control)
  - If classical_bit_c == 1: fold X-init (emit X(c)), then X(b) instead of CX(b,c), X(a) instead of CX(a,c), then CCX(c,a,b) unchanged
- `emit_CQ_UMA(seq, layer, a, b, c)` — identical to existing emit_UMA (no simplification for UMA per locked decision)

Add a static `compute_CQ_layer_count(bits, bin)` function that precisely counts layers:
- For first MAJ (i=0): if bit[0]==0, 0 layers (fully eliminated since carry also |0>); if bit[0]==1, 4 layers (X-init + 2X + CCX)
- For subsequent MAJs (i>=1): if bit[i]==0, 1 layer (CCX only); if bit[i]==1, 4 layers (X-init + 2X + CCX)
- UMA chain: 3*bits layers (unchanged)
- Cleanup: popcount(value) X gates (1 per bit=1 position)

Replace toffoli_CQ_add body (keep 1-bit special case unchanged):
- Call two_complement for binary representation (existing pattern)
- Compute layer count with compute_CQ_layer_count
- Allocate sequence
- Forward sweep: emit_CQ_MAJ for each position (first MAJ has a_known_zero=1, rest have a_known_zero=0)
- Reverse sweep: emit_CQ_UMA for each position (calls standard emit_UMA)
- Cleanup: X(temp[i]) for each bit=1 position
- Add `#ifdef DEBUG` assertion block after UMA chain: verify layer count matches expectation

IMPORTANT — bin[] is MSB-first: bin[0]=MSB, bin[bits-1]=LSB. To get LSB-first bit i, use bin[bits - 1 - i]. This matches the existing code pattern.

**2. `toffoli_cCQ_add(bits, value)` — Inline CDKM cCQ generator:**

Add two static helper functions:
- `emit_cCQ_MAJ(seq, layer, a, b, c, classical_bit_c, a_known_zero, ext_ctrl)`:
  - If classical_bit_c == 0 (temp[i] unconditionally |0>): skip 2 CCX (one control is |0>), emit MCX unless a_known_zero (then entire cMAJ is NOP — carry=|0> makes MCX a NOP too)
  - If classical_bit_c == 1: fold CX-init (emit CX(c, ext_ctrl)), then emit standard cMAJ gates (temp[i] is entangled, cannot simplify per locked decision)
- `emit_cCQ_UMA(seq, layer, a, b, c, ext_ctrl)` — identical to existing emit_cUMA (no simplification)

Add `compute_cCQ_layer_count(bits, bin)`:
- For first cMAJ (i=0): if bit[0]==0, 0 layers (fully eliminated); if bit[0]==1, 4 layers (CX-init + 3 standard cMAJ gates)
- For subsequent cMAJs (i>=1): if bit[i]==0, 1 layer (MCX only); if bit[i]==1, 4 layers (CX-init + 3 standard cMAJ)
- cUMA chain: 3*bits layers (unchanged)
- Cleanup: popcount(value) CX gates

Replace toffoli_cCQ_add body (keep 1-bit special case unchanged).

**3. `toffoli_CQ_add_bk(bits, value)` — Inline BK CLA CQ generator:**

Replace the gate-copy approach with inline BK CLA generation that simplifies Phase A and Phase E:
- Phase A generates: for each position i, if temp[i]=classical_bit=0, skip CCX (one control is |0>) and skip CX (control is |0>); if temp[i]=1, replace CCX with simplified form (emit X(temp[i]) init first, then CX(g[i], self[i]) instead of CCX) and X(self[i]) instead of CX(self[i], temp[i])
- Phases B, C, D: copy from cached QQ BK sequence unchanged (these have quantum-dependent values)
- Phase E: same simplifications as Phase A (mirror, temp qubits restored by Phase D)
- Phase F: unchanged
- Cleanup: X(temp[i]) for each bit=1 position

Use the existing `toffoli_QQ_add_bk(bits)` result to determine layer count for Phases B-D-F, then add simplified Phase A/E layer counts.

**4. `toffoli_cCQ_add_bk(bits, value)` — Inline BK CLA cCQ generator:**

Same approach as CQ BK but using the controlled variant:
- Phase A: for bit=0 positions, skip CCX and CX; for bit=1, fold CX-init and emit standard controlled gates
- Phases B-D: copy from cached cQQ BK sequence unchanged
- Phase E: same simplifications as Phase A
- Phase F: unchanged
- Cleanup: CX(temp[i], ext_ctrl) for each bit=1 position

Keep all function signatures unchanged. No changes to qubit layout. No Python-side changes.

- `_verify_ccq_sub(a_val, value, width)` — controlled subtraction

- `test_cq_sub_exhaustive_width_{1,2,3,4}` — all (a_val, value) pairs for subtraction
- `test_ccq_sub_exhaustive_width_{1,2,3,4}` — controlled subtraction

Mark width-4 exhaustive tests as `@pytest.mark.slow` if they take >5 seconds.
