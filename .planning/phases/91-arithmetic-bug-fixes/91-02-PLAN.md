Create `c_backend/src/ToffoliModReduce.c` implementing modular reduction using the direct-emit pattern.

**Core function (`toffoli_mod_reduce`):**
- Signature: `void toffoli_mod_reduce(circuit_t *circ, const unsigned int *value_qubits, int value_bits, int64_t modulus)`
- Purpose: Reduce a quantum register in-place modulo a classical N. Input assumed to be in range [0, 2N-2] (valid for results of modular addition where both operands < N).
- Algorithm: Single conditional subtraction of N.
  1. Allocate widened comparison ancillae (value_bits + 1 qubits for temp, 1 carry, 1 comparison result qubit).
  2. Copy value to widened temp via CNOT (zero-extend by 1 bit).
  3. Subtract N from widened temp using `toffoli_CQ_add` with negated N (two's complement).
  4. Extract MSB of widened result as sign bit (comparison result: 0 means value >= N, 1 means value < N).
  5. If value >= N (sign bit = 0): subtract N from the original value register using `toffoli_CQ_add` with negated N, controlled by the inverted sign bit. Use X gate on sign bit to invert, then use it as control, then X again to restore.
  6. Uncompute widened temp: add N back to widened temp, un-copy via CNOT.
  7. Free all ancillae.
- Ancilla budget: (value_bits + 1) for widened temp + 1 carry + 1 comparison result = value_bits + 3 total. All freed after use.

**Controlled variant (`toffoli_cmod_reduce`):**
- Signature: `void toffoli_cmod_reduce(circuit_t *circ, const unsigned int *value_qubits, int value_bits, int64_t modulus, unsigned int ext_ctrl)`
- Same algorithm but all operations are controlled by ext_ctrl using the AND-ancilla pattern.

**Modular addition helper (`toffoli_mod_add_cq`):**
- Signature: `void toffoli_mod_add_cq(circuit_t *circ, const unsigned int *value_qubits, int value_bits, int64_t addend, int64_t modulus)`
- Performs value += addend followed by mod_reduce. This provides a single-call API for (value + addend) mod N.
- Controlled variant: `toffoli_cmod_add_cq` with ext_ctrl parameter.

**Header updates:**
- Add all function declarations to `c_backend/include/toffoli_arithmetic_ops.h` under a new "Toffoli Modular Reduction" section (after the Division section added in Plan 91-01).

**CMakeLists.txt:**
- Add `c_backend/src/ToffoliModReduce.c` to the C source file list.

**Important implementation details:**
- Use `toffoli_CQ_add` for subtraction (pass -N as value, since CQ_add handles two's complement internally).
- For the controlled subtraction of N from the original value: emit the CQ subtraction gates individually (cannot use the sequence-based CQ_add in controlled mode without modification). Instead, use the `toffoli_cCQ_add` function which already supports controlled CQ addition. Pass -N as value.
- Follow existing code patterns from ToffoliMultiplication.c for allocator usage, gate emission, and Clifford+T decomposition support.
- Support arbitrary moduli N including even values and powers of 2.

- After calling `qint.__mul__` for the raw product, call `toffoli_mod_reduce` on the result.
