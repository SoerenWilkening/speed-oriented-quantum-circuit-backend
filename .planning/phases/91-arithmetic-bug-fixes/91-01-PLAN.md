  <action>
Create `c_backend/src/ToffoliDivision.c` implementing the restoring division algorithm using the direct-emit pattern (like `toffoli_mul_qq` in ToffoliMultiplication.c).

**Classical divisor divmod (`toffoli_divmod_cq`):**
- Signature: `void toffoli_divmod_cq(circuit_t *circ, const unsigned int *dividend_qubits, int dividend_bits, int64_t divisor, const unsigned int *quotient_qubits, const unsigned int *remainder_qubits)`
- Algorithm: Restoring division. Copy dividend to remainder (via CNOT). For each bit position from MSB to LSB: (1) compare remainder >= divisor_shifted using widened subtraction + MSB check, (2) conditionally subtract divisor_shifted from remainder, (3) conditionally set quotient bit, (4) uncompute comparison ancillae.
- All ancillae (widened temps, carry, comparison result) allocated via `allocator_alloc` and freed via `allocator_free` within the function.
- For comparison: use widened (bits+1) subtraction via `toffoli_QQ_add` with negated divisor to compute `remainder - trial_value`, check MSB (sign bit), then add back to restore remainder. The comparison result (a single qubit) controls the conditional subtraction and quotient bit set.
- Use `emit_ccx_or_clifford_t` helper for CCX gates (same as ToffoliMultiplication.c).
- Division by zero: divisor == 0 should set all quotient bits to 1 (all-ones sentinel) via X gates on quotient qubits, and copy dividend to remainder.

**Quantum divisor divmod (`toffoli_divmod_qq`):**
- Signature: `void toffoli_divmod_qq(circuit_t *circ, const unsigned int *dividend_qubits, int dividend_bits, const unsigned int *divisor_qubits, int divisor_bits, const unsigned int *quotient_qubits, const unsigned int *remainder_qubits)`
- Algorithm: Same restoring division but with quantum-quantum comparison and controlled operations. For each bit position: (1) widen both remainder and shifted-divisor to (bits+1) bits using CNOT copies, (2) subtract widened divisor from widened remainder via `toffoli_QQ_add`, (3) extract MSB as comparison result, (4) controlled subtract/add-back, (5) uncompute widened temps.
- Ancilla budget: up to n+2 ancillae per iteration (widened temps + carry + comparison result), allocated and freed within each iteration.

**Controlled variants (`toffoli_cdivmod_cq`, `toffoli_cdivmod_qq`):**
- Add single-control variants using the AND-ancilla pattern from ToffoliMultiplication.c.
- ext_ctrl parameter gates all operations.

**Header updates:**
- Add function declarations to `c_backend/include/toffoli_arithmetic_ops.h` under a new "Toffoli Division" section.

**CMakeLists.txt:**

**Debug assertions:**

Follow the exact coding patterns from ToffoliMultiplication.c: include the same headers, use the same `allocator_alloc`/`allocator_free` pattern, same gate emission helpers, same LSB-first qubit convention.
