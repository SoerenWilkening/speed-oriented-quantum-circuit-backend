**print_comparison(manual_stats, api_stats) helper:**
- Print formatted table: Metric | Manual | QWalkTree | Delta
- Include qubit_count, gate_count, depth rows
- Delta column shows difference with +/- sign
- Clean, aligned formatting per user decision

**get_api_stats() function:**
1. Import quantum_language as ql, from quantum_language.walk import QWalkTree
2. Compute branching factors from prepare_walk_data() -- extract move_count per level to build branching list
3. Reset chess_walk._walk_compiled_fn = None
4. c = ql.circuit()
5. Define trivial_predicate(node) that returns (ql.qbool(), ql.qbool()) -- always-accept, per user decision
6. tree = QWalkTree(max_depth=MAX_DEPTH, branching=branching_list, predicate=trivial_predicate, max_qubits=200)
7. tree.walk_step() -- build walk step only, no detect() per user decision
8. Return {"qubit_count": c.qubit_count, "gate_count": c.gate_count, "depth": c.depth}

**main() function:**
1. Print header: "QUANTUM CHESS WALK: Manual vs QWalkTree Comparison"
2. Print position info (WK_SQ, BK_SQ, WN_SQUARES, MAX_DEPTH)

**if __name__ == "__main__":** call main()
