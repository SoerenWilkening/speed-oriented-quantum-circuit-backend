    - to_dot() with overlap edge includes dashed arrow labeled "N qubits"

   - Start with `digraph CallGraph {` header
   - Set `rankdir=TB`, `node [shape=box, fontname="Courier"]`
   - Build node_to_group mapping from parallel_groups()
   - If multiple groups exist, wrap each group's nodes in `subgraph cluster_N { style=dotted; label="Group N"; ... }`
   - If only 1 group (or 0), emit nodes without cluster wrapping
   - For each node: label = "func_name\ngates: N\ndepth: N\nqubits: N\nT-count: N" (use \\n for DOT newlines)
   - Sanitize func_name: replace `"` with `\"` in labels
   - For each edge: check edge data type
     - "call" -> solid arrow: `nX -> nY [label="call"];`
     - "overlap" -> dashed: `nX -> nY [style=dashed, label="W qubits"];`
   - Close with `}`
   - Return newline-joined string

   - test_multiple_groups_have_clusters: build 2 disjoint nodes, verify "subgraph cluster_" appears
   - test_single_group_no_cluster: build 2 overlapping nodes, verify "subgraph cluster_" does NOT appear
   - Integration test: @ql.compile(opt=1), call to_dot(), verify valid DOT structure
