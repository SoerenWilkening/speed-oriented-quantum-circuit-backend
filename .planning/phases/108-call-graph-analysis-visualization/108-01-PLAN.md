1. Add `aggregate()` method to CallGraphDAG:
   ```python
   def aggregate(self) -> dict:
   ```
   - If no nodes, return {"gates": 0, "depth": 0, "qubits": 0, "t_count": 0}
   - total_gates = sum(n.gate_count for n in self._nodes)
   - total_t = sum(n.t_count for n in self._nodes)
   - all_qubits = union of all n.qubit_set
   - groups = self.parallel_groups()
   - total_depth = sum(max(self._nodes[idx].depth for idx in group) for group in groups)
   - Return {"gates": total_gates, "depth": total_depth, "qubits": len(all_qubits), "t_count": total_t}
