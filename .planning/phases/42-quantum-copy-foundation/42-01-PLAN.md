2. Get the circuit pointer: `cdef circuit_t *_circuit = <circuit_t*><unsigned long long>_get_circuit()` and `cdef bint _circuit_initialized = _get_circuit_initialized()`.
3. Capture start layer: `start_layer = (<circuit_s*>_circuit).used_layer if _circuit_initialized else 0`.
4. Allocate a fresh result qint with `result = qint(width=self.bits)` — this gives |0> qubits, handles scope registration and creation counter.
5. Apply CNOTs from source to result using the existing `Q_xor` pattern (identical to the first CNOT block in `__xor__` lines 1510-1519):
   - `self_offset = 64 - self.bits`
   - `result_offset = 64 - result.bits`
   - `qubit_array[:self.bits] = result.qubits[result_offset:result_offset + self.bits]`
   - `qubit_array[self.bits:2*self.bits] = self.qubits[self_offset:64]`
   - `arr = qubit_array`
   - `seq = Q_xor(self.bits)`
   - `run_instruction(seq, &arr[0], False, _circuit)`
6. Set layer tracking metadata:
   - `result._start_layer = start_layer`
   - `result._end_layer = (<circuit_s*>_circuit).used_layer if _circuit_initialized else 0`
   - `result.operation_type = 'COPY'`
   - `result.add_dependency(self)`
7. Return `result`.

Declare the necessary cdef variables at the top of the method: `seq` as `sequence_t *`, `arr` as `unsigned int[:]`, `self_offset` / `result_offset` as int, `start_layer` as int, plus the circuit pointer and initialized flag.

**`copy_onto(self, target)` method:**
1. Call `self._check_not_uncomputed()`.

   - `arr = qubit_array`
   - `seq = Q_xor(self.bits)`
   - `run_instruction(seq, &arr[0], False, _circuit)`
5. No layer tracking or dependency on the target — copy_onto is a raw CNOT operation, the caller manages lifecycle.

Place both methods near the existing bitwise operations (after `__xor__` or after `__invert__`), following the same docstring style (NumPy-style with Parameters, Returns, Examples sections).

Add a `copy()` method to the qbool class that preserves the qbool type:

```python
def copy(self):
    """Create a quantum copy of this boolean.

    Returns a new qbool with fresh qubit CNOT-entangled to this boolean's qubit.
    The copy measures to the same value as the source for computational basis states.

    Returns
    -------
    qbool
        New quantum boolean with copied state.

    Examples
    --------
    >>> flag = qbool(True)
    >>> flag_copy = flag.copy()
    >>> flag_copy.width
    1
    """
    # Use qint.copy() for the CNOT logic
    result_qint = qint.copy(self)
    # Wrap in qbool using existing qubit (no new allocation)
    return qbool(create_new=False, bit_list=result_qint.qubits[63:64])
```

Key: Pass `create_new=False` and `bit_list` to reuse the qubits allocated by qint.copy(). This avoids double-allocation while preserving the qbool type. Check that the returned qbool has its layer tracking metadata carried over from the intermediate result_qint — if not, manually copy `_start_layer`, `_end_layer`, `operation_type`, and dependency from result_qint to the qbool result.
