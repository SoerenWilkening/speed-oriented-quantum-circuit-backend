1. **Add gate type constants** at module level (from types.h enum Standardgate_t):

   ```python
   # Gate type constants (from c_backend/include/types.h Standardgate_t)
   _X, _Y, _Z, _R, _H, _Rx, _Ry, _Rz, _P, _M = range(10)
   _SELF_ADJOINT = frozenset({_X, _Y, _Z, _H})
   _ROTATION_GATES = frozenset({_P, _Rx, _Ry, _Rz, _R})
   ```

2. **Add `_gates_cancel(g1, g2)` function**: Returns True if two gate dicts are exact inverses that would cancel to identity. Rules:
   - Must have same target, same num_controls, same controls, same type
   - Self-adjoint gates (X, Y, Z, H) always cancel with themselves
   - Rotation gates (P, Rx, Ry, Rz, R) cancel when angles sum to zero (use `abs(g1['angle'] + g2['angle']) < 1e-12`)
   - Measurement gates never cancel

3. **Add `_gates_merge(g1, g2)` function**: Returns True if two gate dicts can be merged into one. Rules:
   - Must have same target, same num_controls, same controls, same type
   - Only rotation gates can merge (angles add)
   - Self-adjoint and measurement gates cannot merge (they'd cancel, not merge)

4. **Add `_merged_gate(g1, g2)` function**: Returns a new gate dict with merged angle (`g1['angle'] + g2['angle']`). Copy all other fields from g1. If merged angle is effectively zero (abs < 1e-12), return None (gate disappears).

5. **Add `_optimize_gate_list(gates)` function**: Multi-pass optimization:
   ```python
   def _optimize_gate_list(gates):
       """Optimize gate list with multiple passes until stable."""
       prev_count = len(gates) + 1
       optimized = list(gates)
       max_passes = 10  # safety limit
       passes = 0
       while len(optimized) < prev_count and passes < max_passes:
           prev_count = len(optimized)
           passes += 1
           result = []
           for gate in optimized:
               if result and _gates_cancel(result[-1], gate):
                   result.pop()  # Adjacent inverse cancellation
               elif result and _gates_merge(result[-1], gate):
                   merged = _merged_gate(result[-1], gate)
                   if merged is None:
                       result.pop()  # Merged to zero
                   else:
                       result[-1] = merged
               else:
                   result.append(gate)
           optimized = result
       return optimized
   ```

6. **Add `optimize` parameter to `CompiledFunc.__init__`**: Store as `self._optimize`. Default True.

7. **Add stats tracking to `CompiledBlock`**: Add `original_gate_count` slot (int). Set in _capture before optimization.

8. **Wire optimization into `_capture` method**: After building virtual_gates and before creating CompiledBlock:
   ```python
   original_count = len(virtual_gates)
   if self._optimize:
       try:
           virtual_gates = _optimize_gate_list(virtual_gates)
       except Exception:

   ```
   Store original_count in CompiledBlock.

9. **Add stats properties to CompiledFunc**:
   ```python
   @property
   def original_gates(self):
       """Total original gate count across all cache entries."""
       return sum(b.original_gate_count for b in self._cache.values())

   @property
   def optimized_gates(self):
       """Total optimized gate count across all cache entries."""
       return sum(len(b.gates) for b in self._cache.values())

   @property
   def reduction_percent(self):
       """Percentage reduction from optimization."""
       orig = self.original_gates
       if orig == 0:
           return 0.0
       return 100.0 * (1.0 - self.optimized_gates / orig)
   ```
