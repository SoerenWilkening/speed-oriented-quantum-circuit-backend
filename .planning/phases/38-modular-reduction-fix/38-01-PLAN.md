  <name>Task 1: Replace _reduce_mod with Beauregard-style subtract-and-check-sign, fix __sub__ negative handling</name>
  <files>src/quantum_language/qint_mod.pyx</files>
  <action>
  Replace the `_reduce_mod` method (lines 107-131) with the Beauregard-style subtract-and-check-sign pattern. The new algorithm:

  1. **Widen value register by 1 bit** for sign detection: Create a `(value.bits + 1)`-bit qint, copy value bits into it using individual CNOT gates (LSB-aligned, same pattern as `__lt__` in qint.pyx lines 1846-1853). The MSB (extra bit) stays 0 as zero-extension.

  2. **Iterate subtract-check-sign**: For each iteration:
     - Subtract N unconditionally: `widened -= self._modulus`
     - Check sign: `sign = widened[63]` -- MSB at position 63 (right-aligned storage, per `__getitem__` convention where `[63]` accesses the physical MSB qubit)
     - If negative, add N back: `with sign: widened += self._modulus`

  3. **Copy result back** to a value-width qint using CNOT copy (same LSB-aligned pattern).

  4. **Iteration count**: Use `max_subtractions = (max_val - 1) // self._modulus` where `max_val = 1 << value.bits`. Iterate `max(1, max_subtractions)` times. This is a safe upper bound -- each iteration reduces the value by at most N.

  **Important implementation details:**
  - Use `cdef` for C-typed locals (comp_width, max_val, etc.)
  - Access the circuit pointer via `_get_circuit()` / `_get_circuit_initialized()` the same way `__lt__` does
  - The CNOT copy pattern uses `qubit_array[0]` = target, `qubit_array[1]` = source, then `Q_xor(1)` and `run_instruction`
  - Import the C-level symbols already available in the .pyx via cimport: `qubit_array`, `Q_xor`, `run_instruction`, `circuit_t`, `circuit_s`
  - `widened[63]` creates a qbool VIEW (shares physical qubit, no new allocation) -- correct for use as `with` control
  - After the loop, the widened MSB is guaranteed to be 0 (value in [0, N)), so discarding it is safe

  **Also fix `__sub__` method** (lines 191-235): Replace the `is_negative = diff < 0; with is_negative: diff += self._modulus` pattern with the same MSB sign check approach:
  - After `diff = qint.__sub__(self, other)`, the diff register's MSB indicates negativity
  - Use `sign = diff[63]` (MSB as sign indicator)
  - `with sign: diff += self._modulus`
  - This avoids creating a separate comparison qbool that leaks entanglement and also stabilizes qubit layout (fixing extraction instability)

  **Do NOT modify** `__ge__`, `__lt__`, or any comparison operators in qint.pyx -- the fix is isolated to qint_mod.pyx.
