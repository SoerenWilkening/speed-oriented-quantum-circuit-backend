   - `arr[i] += x` (add)

   - `arr[i] *= x` (multiply)

   - `arr[i] <<= x` (left shift)

   - `arr[i] |= x` (bitwise OR)
   - `arr[i] ^= x` (bitwise XOR)

3. **AMUT-03: Multi-dimensional indexing**
   - Create 2D qarray (e.g., shape (2, 3))

   - Verify shape unchanged after operation
