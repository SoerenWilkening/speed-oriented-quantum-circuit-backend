  <name>Task 1: Implement CQ_equal_width function</name>
  <files>Backend/src/IntegerComparison.c, Backend/include/comparison_ops.h</files>
  <action>
Replace the stub CQ_equal_width implementation with a working gate sequence generator.

Algorithm (XOR-based equality check, similar to Python-level implementation):
1. For each bit position i where classical_bit[i] == 0:
   - Apply X gate to quantum bit i (flip it so it should become 1 if originally 0)
2. After step 1: if quantum == classical, all qubits that were different are now |1>
3. Apply multi-controlled X using cascaded Toffoli gates (ccx from gate.h) to set result qubit
   - For n-bit comparison: use n-1 ancilla qubits and cascade of Toffoli gates
   - Pattern from existing code: accumulate AND results into ancilla, final result in qubit[0]
4. Reverse the X gates from step 1 to restore original quantum state (uncompute)

Qubit layout (matches existing conventions from CQ_add/CQ_and):
- [0]: Result qubit (output, set to |1> if equal)
- [1, bits]: Quantum operand to compare (indices 1 through bits inclusive)

Edge cases:
- If value >= 2^bits (unsigned overflow for positive values): return empty sequence
  - Allocate sequence_t but set num_layer=0, used_layer=0
  - Caller can check seq->num_layer == 0 to detect this
- If bits <= 0 or bits > 64: return NULL (invalid width)
- If value < 0: use two_complement() from Integer.h for binary conversion

Memory management:
- Follow CQ_add pattern from IntegerAddition.c (lines 70-127):
  - Allocate sequence_t with malloc
  - Allocate gates_per_layer with calloc
  - Allocate seq with calloc (array of gate_t* pointers)
  - Allocate each layer's gate array with calloc
  - Caller owns returned sequence (must free)
- Use two_complement(value, bits) from Integer.h for binary conversion

Gate creation:
- Use x(gate_t*, target) for X gates (from gate.h)
- Use ccx(gate_t*, target, control1, control2) for Toffoli gates (from gate.h)

Function signature stays the same: sequence_t *CQ_equal_width(int bits, int64_t value)
