**Header changes (circuit_output.h):**
Add after existing declarations:
```c
// Export circuit to OpenQASM 3.0 string (heap-allocated)
// Returns malloc'd string buffer containing valid OpenQASM 3.0
// Caller must free() the returned pointer
// Returns NULL on error (NULL circuit, malloc failure)
char* circuit_to_qasm_string(circuit_t *circ);

// Export circuit to OpenQASM 3.0 file (fixed version)
// Returns 0 on success, -1 on error
int circuit_to_openqasm(circuit_t *circ, const char *path);
```

**Implementation (circuit_output.c):**

Add `#include <math.h>` at top (for fmod, M_PI). Include gate.h if not already included.

1. **Add static helper: `normalize_angle(double theta)`**
   - Return `fmod(theta, 2.0 * M_PI)`, if negative add `2.0 * M_PI`
   - Purpose: normalize rotation angles to [0, 2π)

2. **Add static helper: `_get_control_qubit(gate_t *g, int index)`**
   - If `g->NumControls > MAXCONTROLS && g->large_control != NULL`, return `g->large_control[index]`
   - Otherwise return `g->Control[index]`
   - Note: similar function exists in gate.c as `gate_get_control()` — duplicate here as static to avoid header dependency changes

3. **Add static helper: `_count_measurements(circuit_t *circ)`**
   - Loop over all layers and gates, count gates where `g.Gate == M`
   - Return the count as int

4. **Add static helper: `_export_gate(gate_t *g, char *buffer, size_t buf_size, size_t offset, int *measurement_idx)`**
   - Returns new offset (or (size_t)-1 on error)
   - Handle three branches based on `g->NumControls`:
     - **NumControls == 0 (no controls):** switch on Gate type:
       - X → `"x q[%d];\n"` with Target
       - Y → `"y q[%d];\n"` with Target
       - Z → `"z q[%d];\n"` with Target
       - H → `"h q[%d];\n"` with Target
       - P → `"p(%.17g) q[%d];\n"` with normalize_angle(GateValue), Target
       - Rx → `"rx(%.17g) q[%d];\n"` with normalize_angle(GateValue), Target
       - Ry → `"ry(%.17g) q[%d];\n"` with normalize_angle(GateValue), Target
       - Rz → `"rz(%.17g) q[%d];\n"` with normalize_angle(GateValue), Target
       - M → `"c[%d] = measure q[%d];\n"` with *measurement_idx, Target. THEN increment: `(*measurement_idx)++` after the sprintf call.
       - R → `"reset q[%d];\n"` with Target
       - default → `"// unknown gate %d\n"` with Gate value
     - **NumControls == 1:** prefix `"c"` before gate name:
       - X → `"cx q[%d], q[%d];\n"` with _get_control_qubit(g,0), Target
       - Z → `"cz q[%d], q[%d];\n"`
       - Y → `"cy q[%d], q[%d];\n"`
       - H → `"ch q[%d], q[%d];\n"`
       - P → `"cp(%.17g) q[%d], q[%d];\n"` with normalize_angle(GateValue)
       - Rx → `"crx(%.17g) q[%d], q[%d];\n"`
       - Ry → `"cry(%.17g) q[%d], q[%d];\n"`
       - Rz → `"crz(%.17g) q[%d], q[%d];\n"`
       - M/R → skip (not meaningfully controlled)
     - **NumControls == 2:** prefix `"cc"`:
       - X → `"ccx q[%d], q[%d], q[%d];\n"` with _get_control_qubit(g,0), (g,1), Target
       - For other gates: use `"ctrl(2) @ gatename q[ctrl0], q[ctrl1], q[target];\n"` pattern
     - **NumControls > 2:** use `"ctrl(%d) @ "` syntax:
       - Write `"ctrl(%d) @ "` with NumControls
       - Write base gate name (e.g., `"x "`, `"p(%.17g) "`, etc.)
       - Write all control qubits `"q[%d], "` using `_get_control_qubit(g, i)` for i=0..NumControls-1
       - Write target `"q[%d];\n"` with Target
   - **Buffer safety:** Before each sprintf, check if `offset > buf_size - 200`. If so, return `(size_t)-1` to signal reallocation needed.

5. **Implement `circuit_to_qasm_string(circuit_t *circ)`:**
   - If `circ == NULL`, return NULL
   - `int num_measurements = _count_measurements(circ);`
   - `size_t buf_size = 512 + (circ->used * 100);`
   - `char *buffer = malloc(buf_size);` — if NULL, return NULL
   - `size_t offset = 0;`
   - Write header: `"OPENQASM 3.0;\ninclude \"stdgates.inc\";\n\nqubit[%d] q;\n"` with `circ->used_qubits + 1`
   - If `num_measurements > 0`: `"bit[%d] c;\n"` with num_measurements
   - Write `"\n"` separator
   - `int measurement_idx = 0;`
   - Loop over all layers and gates:
     - Call `_export_gate()` for each gate
     - **Realloc retry loop:** If `_export_gate` returns `(size_t)-1`:
       1. `buf_size *= 2;`
       2. `char *tmp = realloc(buffer, buf_size);`
       3. If `tmp == NULL`: `free(buffer); return NULL;`
       4. `buffer = tmp;`
       5. Retry the SAME gate (do not advance to next gate)
   - Null-terminate: `buffer[offset] = '\0';`
   - Return buffer

**Important implementation notes:**
- Use `sprintf(buffer + offset, ...)` and add return value to offset
- MAXCONTROLS is 2 (defined in types.h) — this is the threshold for large_control
- Measurement syntax: use `"c[%d] = measure q[%d];\n"` consistently — this is valid OpenQASM 3.0 assignment syntax. Do NOT use the arrow syntax `measure q -> c`.
