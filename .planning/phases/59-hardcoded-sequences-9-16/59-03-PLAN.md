**CQ_add() modification** (in the function starting at line 24):

Before the existing cache check (`if (precompiled_CQ_add_width[bits] != NULL)`), add:

```c
    // Use hardcoded template for widths 1-16 (allocates on first call, caches)
    if (bits <= HARDCODED_MAX_WIDTH && precompiled_CQ_add_width[bits] == NULL) {
        sequence_t *hardcoded = get_hardcoded_CQ_add(bits);
        if (hardcoded != NULL) {
            precompiled_CQ_add_width[bits] = hardcoded;
        }
    }
```

This populates the cache with the hardcoded template on first call, so the existing cache path handles it from there. The rotation angle injection happens in the existing `if (precompiled_CQ_add_width[bits] != NULL)` block which already updates `GateValue` fields.

Insert this block AFTER the rotation computation (after `free(bin)`) but BEFORE the existing cache check. This way:
1. Rotations are computed (needed for angle injection)
2. Hardcoded template is loaded into cache (first call only)
3. Existing cache path injects angles and returns
