Verify output has 3 gates in reversed order: first gate is P type with angle=-0.5 on target 0, second is H type on target 1, third is X type on target 0.

2. `test_inverse_negates_rotation_angles()`: For each rotation gate type (`_P`, `_Rx`, `_Ry`, `_Rz`, `_R`), create a gate with angle=1.23 using the `_gate` helper. Call `_adjoint_gate()`. Assert the returned gate has `angle == -1.23` (within floating-point tolerance). Assert the type is unchanged.

3. `test_inverse_self_adjoint_unchanged()`: For each self-adjoint gate type (`_X`, `_Y`, `_Z`, `_H`), create a gate with `_gate(type, 0)`. Call `_adjoint_gate()`. Assert the returned gate has same type and `angle == 0.0`.

4. `test_inverse_measurement_raises()`: Create `_gate(_M, 0)`. Call `_adjoint_gate()`. Assert it raises `ValueError`. Assert "measurement" (case-insensitive) is in the error message.

5. `test_inverse_empty_function()`: `ql.circuit()`. `@ql.compile` a noop function `def noop(x): return x`. Call `noop(qint)` to capture. Call `noop.inverse()` to get the inverse callable. Call the inverse on a new qint. Assert the result is the input qint (in-place return). Assert the inverse is an `_InverseCompiledFunc` instance.

6. `test_inverse_round_trip()`: `ql.circuit()`. `@ql.compile` a function. Call it to populate cache. Assert `fn.inverse()` is an `_InverseCompiledFunc`. Assert `fn.inverse().inverse() is fn` (identity round-trip).

7. `test_inverse_replays_adjoint_gates()`: `ql.circuit()`. `@ql.compile` `add_one(x): x += 1; return x`. Call it to capture. Get the inverse. Record start layer. Call inverse on a new qint. Record end layer. Extract gates. Verify gates were produced (count > 0). Verify the inverse call's gate count equals the original block's gate count (inverse has same number of gates as forward).

8. `test_inverse_preserves_controls()`: Create a gate `_gate(_P, 0, angle=0.5, controls=[1, 2])`. Call `_adjoint_gate()`. Assert the adjoint gate has `controls == [1, 2]` and `num_controls == 2`. Assert `angle == -0.5`.

1. `test_nesting_inner_gates_in_outer_capture()`: `ql.circuit()`. Define inner: `@ql.compile` `add_one(x): x += 1; return x`. Define outer: `@ql.compile` `add_two(x): x = add_one(x); x = add_one(x); return x` (outer calls inner twice). Use call counters for both. Call `add_one` once to populate its cache. Then call `add_two(ql.qint(3, width=4))` -- this captures outer (which internally replays inner twice). Assert outer call counter is 1. Assert `add_two` produced gates (layer advanced). Call `add_two` again on a different qint -- assert outer call counter stays at 1 (replay, no re-execution). Assert inner call counter stayed at 1 throughout.

2. `test_nesting_depth_limit()`: Import `_MAX_CAPTURE_DEPTH` from `quantum_language.compile`. `ql.circuit()`. Create a module-level list `fn_ref = [None]`. Define `@ql.compile` function `recursive_fn(x)` that increments x and calls `fn_ref[0](x)`. Set `fn_ref[0] = recursive_fn`. Call `recursive_fn(ql.qint(0, width=4))`. Assert it raises `RecursionError`. Assert "nesting depth" (case-insensitive) is in the error message. Use `pytest.raises(RecursionError)`.

3. `test_nesting_inner_return_value_usable()`: `ql.circuit()`. Inner: `@ql.compile` `make_new(x): return x + ql.qint(1, width=x.width)` (returns new qint). Outer: `@ql.compile` `add_and_increment(x): r = make_new(x); r += 1; return r`. Call inner once to cache it. Call outer with a qint. Assert result is a qint with correct width. Call outer again (replay). Assert replay result width is correct and subsequent operations work (layer advances after `result += 1`).
