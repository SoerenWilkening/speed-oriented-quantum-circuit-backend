<objective>
Implement auto-uncomputation of temp ancilla qubits in compiled functions when `ql.option("qubit_saving")` is active.

Purpose: When qubit-saving mode is on, compiled functions automatically clean up internal scratch qubits after the forward call, preserving only return value qubits. This reduces qubit usage without manual f.inverse(x) calls for temp ancillas.

Output: Modified `compile.py` with auto-uncompute logic, cache key extension, input side-effect detection, and updated _AncillaInverseProxy for post-auto-uncompute inverse support.
</objective>
