Implementation:
1. Import CompiledFunc from quantum_language.compile
2. Create fixture parametrized over [1, 2, 3] with ids ["opt1", "opt2", "opt3"]:
   ```python
   @pytest.fixture(params=[1, 2, 3], ids=["opt1", "opt2", "opt3"])
   def opt_level(request, monkeypatch):
       level = request.param
       original_init = CompiledFunc.__init__
       def patched_init(self, func, **kwargs):
           if 'opt' not in kwargs:
               kwargs['opt'] = level
           original_init(self, func, **kwargs)
       monkeypatch.setattr(CompiledFunc, '__init__', patched_init)
       return level
   ```

3. Key behaviors of the fixture:
   - Only overrides opt when the test doesn't explicitly pass opt=N
   - Tests that explicitly set opt (like opt=2 specific tests) are NOT affected
   - parametric=True + opt=2 tests that explicitly test ValueError are NOT affected (they set opt explicitly)
   - Returns the current opt level so tests can skip/xfail conditionally

IMPORTANT edge cases:
- Tests that explicitly set opt=N in @ql.compile(opt=N) should still work -- the monkeypatch only sets opt when not already provided

Handle this by: (a) detecting parametric in the patched init and skipping override when parametric=True and level==2, OR (b) marking parametric tests to skip opt=2 runs

def patched_init(self, func, **kwargs):
    if 'opt' not in kwargs:
        # Don't force opt=2 on parametric functions (ValueError by design)
        if not (level == 2 and kwargs.get('parametric', False)):
            kwargs['opt'] = level
    original_init(self, func, **kwargs)
