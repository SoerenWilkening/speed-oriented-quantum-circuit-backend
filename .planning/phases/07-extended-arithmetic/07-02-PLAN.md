    Update Python comparison operators to use new C functions or improved pattern:

    The current __gt__, __lt__ etc. work but destroy operands. Improve by:
    1. Allocating ancilla for subtraction result
    2. Copying self to ancilla
    3. Subtracting other from ancilla
    4. Extracting MSB from ancilla as result

    ```python
    def __lt__(self, other):
        """Less-than comparison: self < other

        Returns qbool result. Preserves input operands.
        Uses subtraction and MSB check.
        """
        # Determine comparison width
        if type(other) == int:
            comp_bits = self.bits
            other_qint = qint(other, width=comp_bits)
        elif type(other) == qint:
            comp_bits = max(self.bits, (<qint>other).bits)
            other_qint = other
        else:
            raise TypeError("Comparison requires qint or int")

        # Allocate ancilla for subtraction
        temp = qint(0, width=comp_bits)

        # Copy self to temp (temp = self via XOR from 0)
        temp ^= self

        # Subtract other from temp (temp = self - other)
        temp -= other_qint

        # Extract MSB (sign bit)
        # MSB is at index (64 - comp_bits) for right-aligned storage
        msb = temp[64 - comp_bits]

        # Result: if MSB=1 (negative), self < other is True
        result = qbool()
        # Copy MSB to result
        result ^= msb

        return result
    ```

    **__eq__ implementation using XOR-based equality:**

    ```python
    def __eq__(self, other):
        """Equality comparison: self == other

        Uses optimized XOR-based circuit.
        Returns qbool result. Preserves input operands.
        """
        if type(other) == int:
            comp_bits = self.bits
            other_qint = qint(other, width=comp_bits)
        elif type(other) == qint:
            comp_bits = max(self.bits, (<qint>other).bits)
            other_qint = other
        else:
            raise TypeError("Comparison requires qint or int")

        # XOR self and other into temp
        temp = self ^ other_qint

        # Check if temp == 0 (all bits zero)
        # If any bit is 1, not equal
        result = qbool(True)  # Assume equal

        # For each bit, if it's 1, set result to False
        for i in range(comp_bits):
            bit = temp[64 - comp_bits + i]
            with bit:
                result = ~result  # Toggle result if bit is 1

        return result
    ```
