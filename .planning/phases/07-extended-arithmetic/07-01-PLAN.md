       ```c
       sequence_t *QQ_mul(int bits) {
           if (bits < 1 || bits > 64) return NULL;
       ```

    2. Add width-parameterized cache:
       ```c
       // At file scope
       sequence_t *precompiled_QQ_mul_width[65] = {NULL};

       // In function
       if (precompiled_QQ_mul_width[bits] != NULL)
           return precompiled_QQ_mul_width[bits];
       ```

    3. Replace all INTEGERSIZE references with bits:
       - `mul->num_layer = bits * (2 * bits + 6) - 1;`
       - `QFT(mul, bits);`
       - All loops: `for (int bit = bits - 1; ...)` etc.
       - `QFT_inverse(mul, bits);`

    4. Cache result: `precompiled_QQ_mul_width[bits] = mul;`

    5. Maintain backward compat: if bits == INTEGERSIZE, also set legacy precompiled_QQ_mul

    Note: Helper functions CP_sequence, CX_sequence, all_rot also need bits parameter.
    Create _width variants or add bits parameter to existing functions.
  </action>
