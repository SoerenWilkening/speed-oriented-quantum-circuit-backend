Implement the three remaining BK CLA variants in ToffoliAddition.c, replacing the NULL-returning stubs.

**Approach: Sequence-copy for all three variants.** This is the committed strategy -- do NOT use inline emission. The CQ and controlled variants build new sequences by copying gates from the cached QQ BK sequence, with modifications as needed.

**1. toffoli_CQ_add_bk(bits, value) -- BK CQ adder (sequence-copy)**

The CQ qubit layout maps directly to QQ layout:
- CQ: [0..n-1]=temp (a-register), [n..2n-1]=self (b-register), [2n..end]=CLA ancilla
- QQ: [0..n-1]=a, [n..2n-1]=b, [2n..end]=CLA ancilla

Since the qubit indices match exactly, the CQ function can:
1. Convert value to binary via `two_complement(value, bits)`
2. Count x_count (number of set bits in classical value)
3. Get the cached QQ BK sequence: `seq_qq = toffoli_QQ_add_bk(bits)`
4. Allocate a NEW sequence with layers = 2*x_count + seq_qq->num_layer
5. **X-init phase:** For each set classical bit i, emit X(temp[i]) -- i.e., X(qubit i)
6. **Copy QQ gates:** For each layer in seq_qq, copy all gates into the new sequence (same qubit indices, no offset needed)
7. **X-cleanup phase:** Same X gates as X-init phase (X is self-inverse)
8. Return the new sequence

For bits == 1: follow existing CQ 1-bit pattern (X gate or identity based on value).
For bits < 2 when QQ returns NULL: return NULL (fallback to RCA).

OWNERSHIP: Caller must free (NOT cached, value-dependent). Same as existing toffoli_CQ_add.

**2. toffoli_cQQ_add_bk(bits) -- Controlled BK QQ adder (sequence-copy with control injection)**

Every gate in the QQ BK sequence gets an additional control qubit (ext_ctrl):
- CCX(target, ctrl1, ctrl2) becomes MCX(target, [ctrl1, ctrl2, ext_ctrl]) -- 3-control gate
- CX(target, control) becomes CCX(target, control, ext_ctrl) -- 2-control gate

Qubit layout:
```
[0..n-1]              = a-register
[n..2n-1]             = b-register
[2n..2n+anc-1]        = CLA ancilla (same count as uncontrolled BK)
[2n+anc]              = ext_ctrl (external control qubit)
```

Implementation:
1. Get cached QQ BK sequence: `seq_qq = toffoli_QQ_add_bk(bits)`
2. Allocate new sequence with same layer count as seq_qq
3. For each gate in seq_qq:
   - If CCX (NumControls==2): emit MCX with 3 controls [ctrl1, ctrl2, ext_ctrl]
   - If CX (NumControls==1): emit CCX with controls [control, ext_ctrl]
   - If X (NumControls==0): emit CX with control ext_ctrl
4. Cache in precompiled_toffoli_cQQ_add_bk[bits]

ext_ctrl qubit index = 2*bits + bk_cla_ancilla_count(bits).

For bits < 2: return NULL (let dispatch use CDKM 1-bit controlled).

OWNERSHIP: Cached -- DO NOT FREE.

**3. toffoli_cCQ_add_bk(bits, value) -- Controlled BK CQ adder (sequence-copy)**

Combines CQ (X-init/cleanup) with controlled gates:
1. Convert value to binary
2. Get cached controlled QQ BK sequence: `seq_cqq = toffoli_cQQ_add_bk(bits)`
3. Allocate new sequence with layers = 2*x_count + seq_cqq->num_layer
4. **Controlled X-init:** For each set classical bit i, emit CX(target=temp[i], control=ext_ctrl) instead of X
5. **Copy controlled QQ gates:** Copy all gates from seq_cqq (qubit indices match)
6. **Controlled X-cleanup:** Same CX gates as controlled X-init

ext_ctrl qubit index = 2*bits + bk_cla_ancilla_count(bits) (same position as in cQQ).

OWNERSHIP: Caller must free (NOT cached, value-dependent).
