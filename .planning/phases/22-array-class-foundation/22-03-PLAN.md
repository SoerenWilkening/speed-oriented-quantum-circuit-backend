   - If value doesn't fit in width, warn then truncate:
     ```python
     import warnings
     max_val = max(abs(v) for v in values)
     if max_val.bit_length() > width:
         warnings.warn(f"Values exceed {width}-bit range, will be truncated", UserWarning)
     ```
   - Pass width to qint constructor: `qint(v, width=width)`

2. **dtype parameter:**
   - If `dtype=qbool`, create qbool objects instead of qint
   - For qbool, width is always 1
   - Convert values to bool before passing to qbool
   - Validate dtype is qint or qbool, raise TypeError otherwise

3. **Homogeneity validation:**
   - If data contains existing qint/qbool objects, validate all same type
   - Mixed types raise ValueError: "Array must be homogeneous: cannot mix qint and qbool"

4. **Update __init__ signature:**
   ```python
   def __init__(self, data=None, *, width=None, dtype=None, dim=None):
   ```
   The `*` forces keyword-only for width/dtype/dim (NumPy-style API).

   ```python
   if isinstance(data, np.ndarray):
       # Use NumPy dtype to determine width

       # Convert to nested Python list for shape detection
       data = data.tolist()
   ```

2. **Dimension-based construction (dim parameter):**
   ```python
   if dim is not None:
       if data is not None:
           raise ValueError("Cannot specify both data and dim")
       if dtype is None:
           dtype = qint  # Default to qint

       # Handle int or tuple
       if isinstance(dim, int):
           shape = (dim,)
       else:
           shape = tuple(dim)

       # Calculate total elements
       total = 1
       for d in shape:
           total *= d

       # Create elements (default value 0)
       if dtype == qbool:
           elements = [qbool() for _ in range(total)]
           self._width = 1
       else:
           elements = [qint(0, width=width or INTEGERSIZE) for _ in range(total)]
           self._width = width or INTEGERSIZE

       self._elements = elements
       self._shape = shape
       self._dtype = dtype
       return  # Skip normal data processing
   ```

3. **Update construction flow:**
   - Check `dim` first (dimension-based)
   - Then check `data` type (NumPy or list)
   - Handle each appropriately
