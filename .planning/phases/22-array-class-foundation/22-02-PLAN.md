  <files>src/quantum_language/qarray.pyx</files>
  <action>
Implement complete `__getitem__(self, key)`:

1. **Single int index (flattened access):**
   ```python
   if isinstance(key, int):
       if key < 0:
           key += len(self._elements)
       if not 0 <= key < len(self._elements):
           raise IndexError(f"Index {key} out of bounds for array with {len(self._elements)} elements")
       return self._elements[key]
   ```

2. **Slice (returns view):**
   ```python
   elif isinstance(key, slice):
       start, stop, step = key.indices(len(self._elements))
       indices = range(start, stop, step)
       return self._create_view([self._elements[i] for i in indices], shape=(len(indices),))
   ```

3. **Tuple (multi-dimensional):**
   ```python
   elif isinstance(key, tuple):
       return self._handle_multi_index(key)
   ```

4. **Helper `_multi_to_flat(self, indices)`:**
   Convert (i, j, ...) to flat index for row-major storage.
   ```python
   # shape (2, 3), index (1, 2) -> 1*3 + 2 = 5
   flat = 0
   stride = 1
   for dim in reversed(range(len(self._shape))):
       idx = indices[dim]
       if idx < 0:
           idx += self._shape[dim]  # Handle negative
       flat += idx * stride
       stride *= self._shape[dim]
   return flat
   ```

5. **Helper `_handle_multi_index(self, key)`:**
   - If all ints: return single element via _multi_to_flat
   - If any slice: collect indices, return view with new shape
   - Handle column slice like `arr[:, 0]` - iterate rows, pick column

6. **Classmethod `_create_view(cls, elements, shape)`:**
   ```python
   @classmethod
   def _from_elements(cls, elements, shape, dtype, width):
       arr = cls.__new__(cls)
       arr._elements = elements  # Shared reference (view)
       arr._shape = shape
       arr._dtype = dtype
       arr._width = width
       return arr
   ```

Note: Views share qint object references, not copies.
