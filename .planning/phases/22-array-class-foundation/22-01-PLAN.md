   - `_flatten(data)`: Flatten nested lists to 1D

   - `width`: Returns self._width
   - `dtype`: Returns self._dtype

6. **Required Sequence methods:**
   - `__len__`: Return len(self._elements)
   - `__getitem__`: For now, just handle single int index returning element
