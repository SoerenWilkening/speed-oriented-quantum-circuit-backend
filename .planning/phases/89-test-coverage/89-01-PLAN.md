  Create `tests/python/test_c_backend.py` with subprocess wrappers for ALL C tests in `tests/c/`.

  **Structure:**

  ```python
  """Subprocess wrappers for C backend unit tests.

  Compiles and runs C tests from tests/c/ directory via make.
  Shows stdout/stderr only on failure for clean output.
  Skips all tests if no C compiler (gcc/cc/clang) is available.
  """
  import os
  import shutil
  import subprocess
  import pytest

  C_TEST_DIR = os.path.join(os.path.dirname(__file__), "..", "c")

  def _has_compiler():
      """Check if a C compiler is available."""
      for compiler in ["gcc", "cc", "clang"]:
          if shutil.which(compiler):
              return True
      return False

  def _compile_and_run(target, timeout_compile=120, timeout_run=60):
      """Compile a C test target and run it.

      Args:
          target: Make target name (e.g., 'test_allocator_block')
          timeout_compile: Compilation timeout in seconds
          timeout_run: Execution timeout in seconds

      Raises:
          pytest.fail on compilation or execution failure
      """
      # Clean first to ensure fresh build
      subprocess.run(["make", "-C", C_TEST_DIR, "clean"],
                     capture_output=True, timeout=30)

      # Compile
      result = subprocess.run(
          ["make", "-C", C_TEST_DIR, target],
          capture_output=True, text=True, timeout=timeout_compile
      )
      if result.returncode != 0:
          pytest.fail(f"Compilation of {target} failed:\n{result.stderr}\n{result.stdout}")

      # Run
      executable = os.path.join(C_TEST_DIR, target)
      result = subprocess.run(
          [executable],
          capture_output=True, text=True, timeout=timeout_run
      )
      if result.returncode != 0:
          pytest.fail(
              f"C test {target} failed (exit code {result.returncode}):\n"
              f"stdout:\n{result.stdout}\n"
              f"stderr:\n{result.stderr}"
          )

  # Skip all tests if no C compiler
  pytestmark = pytest.mark.skipif(
      not _has_compiler(),
      reason="C compiler (gcc/cc/clang) not found - skipping C backend tests"
  )
  ```

  **Test functions (one per C test binary):**

  1. `test_c_allocator_block()` -- target: `test_allocator_block`
  2. `test_c_reverse_circuit()` -- target: `test_reverse_circuit`
  3. `test_c_comparison()` -- target: `test_comparison`
  4. `test_c_hot_path_add()` -- target: `test_hot_path_add`
  5. `test_c_hot_path_mul()` -- target: `test_hot_path_mul`
  6. `test_c_hot_path_xor()` -- target: `test_hot_path_xor`

  Each function: docstring explaining what the C test covers, calls `_compile_and_run(target)`.

  **Key decisions per CONTEXT.md:**

  - Show stdout + stderr only on failure
  - pytest.skip() if no compiler found
  - Use existing Makefile targets
