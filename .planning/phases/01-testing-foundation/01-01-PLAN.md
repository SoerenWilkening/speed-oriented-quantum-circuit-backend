<task type="auto">
  <name>Task 2: Configure pre-commit hooks with Ruff and clang-format</name>
  <files>.pre-commit-config.yaml, pyproject.toml, .clang-format</files>
  <action>
1. Create pyproject.toml with:
   - [project] section with name="quantum-assembly", version="0.1.0", requires-python=">=3.11"
   - [tool.ruff] section:
     - line-length = 100
     - target-version = "py311"
   - [tool.ruff.lint] section:
     - select = ["E", "F", "W", "I", "B", "C4", "UP"]  # pycodestyle, pyflakes, warnings, isort, bugbear, comprehensions, pyupgrade
     - ignore = ["E501"]  # line length handled separately
   - [tool.ruff.format] section for formatting rules

2. Create .pre-commit-config.yaml with:
   - repos:
     - repo: https://github.com/astral-sh/ruff-pre-commit
       rev: v0.9.0  # Use stable version
       hooks:
         - id: ruff
           args: [--fix]
         - id: ruff-format
     - repo: https://github.com/pre-commit/mirrors-clang-format
       rev: v19.1.6
       hooks:
         - id: clang-format
           types_or: [c, c++]

3. Create .clang-format with:
   - BasedOnStyle: LLVM
   - IndentWidth: 4
   - ColumnLimit: 100
   - BreakBeforeBraces: Attach
   - AllowShortFunctionsOnASingleLine: Empty
  </action>

  </verify>
  <done>.pre-commit-config.yaml has Ruff and clang-format hooks, pyproject.toml has Ruff config, .clang-format uses LLVM style</done>
