Create `src/quantum_language/grover.py` implementing the `grover()` function. This is a composition layer -- every component already exists.

**Function signature:**
```python
def grover(oracle, *, width=None, widths=None, m=1, iterations=None):
```

**Internal helpers to implement:**

1. `_get_oracle_func(oracle)` -- Extract the original function from an oracle wrapper:
   - If `GroverOracle`: return `oracle._original_func`
   - If `CompiledFunc`: return `oracle._func`
   - If plain callable: return `oracle`
   - Otherwise: raise `TypeError`

2. `_get_quantum_params(func)` -- Introspect oracle function to find quantum parameter names:
   - Use `inspect.signature(func)` on the original function
   - Each parameter annotated as `qint` (or missing annotation) is a quantum parameter
   - Return list of parameter names
   - Import qint locally to check annotation: `from .qint import qint`

3. `_resolve_widths(param_names, width, widths)` -- Resolve register widths:
   - If `widths` provided (list): validate len matches param count, return widths
   - If `width` provided (int): return `[width] * len(param_names)`
   - If neither: raise `ValueError("width or widths required")`
   - If both provided and inconsistent: raise `ValueError`

4. `_grover_iterations(N, M)` -- Calculate optimal iteration count:
   - Formula: `int(math.floor(math.pi / 4 * math.sqrt(N / M) - 0.5))`
   - Edge cases: M <= 0 or M >= N: return 0
   - Uses `math.floor`, `math.pi`, `math.sqrt`

5. `_ensure_oracle(oracle)` -- Ensure oracle is GroverOracle:
   - If already `GroverOracle`: return as-is
   - Otherwise: wrap with `grover_oracle(oracle)` (auto-compiles if needed)

6. `_apply_hadamard_layer(registers)` -- Apply H gate to every qubit in all registers:
   - Use `_collect_qubits(*registers)` from diffusion module to get qubit indices
   - Call `emit_h(q)` for each qubit
   - **CRITICAL: Use emit_h, NOT branch(0.5).** Research confirmed Ry(pi/2)^2 != I while H^2 = I. The H-sandwich in Grover iterations requires actual Hadamard gates.

7. `_simulate_single_shot(qasm_str, register_widths)` -- Run Qiskit simulation:
   - `import qiskit.qasm3` and `from qiskit import transpile` and `from qiskit_aer import AerSimulator`
   - Parse QASM: `circuit = qiskit.qasm3.loads(qasm_str)`
   - Add measurements if none: `if not circuit.cregs: circuit.measure_all()`
   - Create simulator: `AerSimulator(max_parallel_threads=4)`
   - Run: `sim.run(transpile(circuit, sim), shots=1).result()`
   - Get bitstring: `list(result.get_counts().keys())[0]`
   - Parse bitstring into register values using `_parse_bitstring`

8. `_parse_bitstring(bitstring, register_widths)` -- Parse Qiskit bitstring into register values:
   - Qiskit little-endian: q[0] is rightmost bit in bitstring
   - Search registers allocated first: q[0..w1-1], q[w1..w1+w2-1], etc.
   - Extract rightmost bits for register 1, next bits for register 2, etc.
   - Return tuple of integer values

**Main `grover()` function flow:**
1. `_ensure_oracle(oracle)` -- wrap if needed
2. `_get_quantum_params(_get_oracle_func(oracle))` -- count registers
3. `_resolve_widths(params, width, widths)` -- determine register widths
4. Calculate N = product of 2^w for each register width
5. Calculate iterations: `_grover_iterations(N, m)` or use explicit `iterations` param
6. Build circuit:
   a. `ql.circuit()` -- fresh circuit
   b. `ql.option("fault_tolerant", True)` -- Toffoli backend for oracle comparisons
   c. Create registers: `[ql.qint(0, width=w) for w in register_widths]`
   d. Initialize superposition: `reg.branch(0.5)` for each register (branch(0.5) on |0> is correct)
   e. For each of k iterations:
      - `oracle(*registers)` -- phase flip marked states
      - `_apply_hadamard_layer(registers)` -- H^n
      - `ql.diffusion(*registers)` -- S_0 = X-MCZ-X
      - `_apply_hadamard_layer(registers)` -- H^n
7. Export QASM: `ql.to_openqasm()`
8. Simulate: `_simulate_single_shot(qasm, register_widths)`
9. Return: `(*values, k)` -- flat tuple with iterations last
